# Pregunta

María tiene $150 pesos mexicanos. Si compra 3 tacos al pastor a $25 pesos cada uno, ¿cuánto dinero le quedará?

# Opciones

- [ ] A) $50 pesos
- [ ] B) $60 pesos
- [x] C) $75 pesos
- [ ] D) $100 pesos

# Explicación

Para resolver este problema:

1. **Calcular el gasto total**: 3 tacos × $25 pesos = $75 pesos
2. **Restar del total**: $150 pesos - $75 pesos = **$75 pesos**

La respuesta correcta es **C) $75 pesos**.

**¿Por qué las otras opciones son incorrectas?**
- A) $50 pesos: Error al calcular 150 - 100
- B) $60 pesos: Error al calcular 3 × 30 = 90, luego 150 - 90
- D) $100 pesos: Error al calcular 150 - 50 (pensando que cada taco cuesta $16.67)

## Pregunta 1 (Nivel: 4 - Análisis)
**ID:** "CO-PREU-UIS-001-v1"
### Enunciado
Dada la función f(x) = x³ - 3x + 2, ¿en qué intervalo la función es estrictamente decreciente?
### Opciones
- [ ] A) (-∞, -1)
- [x] B) (-1, 1)
- [ ] C) (1, ∞)
- [ ] D) (-∞, 1)
### Explicación Pedagógica
Análisis de la primera derivada f'(x) = 3x² - 3. La función decrece donde f'(x) < 0. 3(x² - 1) < 0 -> x² < 1 -> -1 < x < 1.

## Pregunta 2 (Nivel: 5 - Evaluación)
**ID:** "CO-PREU-UIS-001-v2"
### Enunciado
Determine el área de la región encerrada entre las curvas y = x² e y = 2x.
### Opciones
- [ ] A) 2/3
- [x] B) 4/3
- [ ] C) 1/2
- [ ] D) 1/3
### Explicación Pedagógica
Puntos de intersección: x² = 2x -> x(x-2) = 0 -> x=0, x=2. Integral de (2x - x²) de 0 a 2 = [x² - x³/3] de 0 a 2 = (4 - 8/3) - 0 = 12/3 - 8/3 = 4/3.

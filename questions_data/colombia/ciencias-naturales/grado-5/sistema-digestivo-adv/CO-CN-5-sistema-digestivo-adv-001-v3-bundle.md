## Pregunta 1 (Muy Fácil A - Dificultad 1)

**ID:** `CO-CN-5-sistema-digestivo-adv-001-v1`

### Enunciado

¿Cuál es la función principal del **Sistema Digestivo**?

### Opciones

- [x] A) Transformar los alimentos en nutrientes que el cuerpo puede usar
- [ ] B) Bombear sangre
- [ ] C) Respirar oxígeno
- [ ] D) Mover los músculos

### Explicación Pedagógica

La digestión es como una fábrica que desarma la comida para sacar lo útil (nutrientes).

### Opciones

- [x] A) El Esófago
- [ ] B) La Tráquea
- [ ] C) El Intestino Delgado

¿Qué sucede con los alimentos si el Páncreas deja de producir sus enzimas?

### Opciones

- [x] A) No se podrían digerir bien las grasas, proteínas ni carbohidratos en el intestino

¿Cómo se relaciona el Sistema Digestivo con el Sistema Circulatorio?

### Opciones

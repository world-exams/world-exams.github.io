# Pregunta Base: Las Máquinas Simples: Multiplicando nuestra fuerza

> **Fuente:** Derechos Básicos de Aprendizaje (DBA) Colombia
> **Tema:** Máquinas simples

---

## Pregunta 1 (Muy Fácil A - Dificultad 1)

**ID:** `CO-CN-4-maquinas-simples-001-v1`

### Enunciado

¿Cuál es el objetivo principal de usar una máquina simple?

### Opciones

- [x] A) Realizar un trabajo aplicando menos fuerza o de forma más cómoda
- [ ] B) Hacer que los objetos se vean más bonitos
- [ ] C) Gastar más energía para hacer ejercicio
- [ ] D) Que los objetos se rompan más rápido

### Explicación Pedagógica

Las máquinas simples cambian la dirección o la magnitud de una fuerza, permitiendo que movamos cosas pesadas con menos esfuerzo.

## Pregunta 2 (Muy Fácil B - Dificultad 1)

**ID:** `CO-CN-4-maquinas-simples-001-v2`

### Enunciado

¿Cómo llamamos a la máquina simple que consiste en una rampa para subir objetos pesados a una altura?

### Opciones

- [x] A) Plano inclinado
- [ ] B) Palanca
- [ ] C) Polea
- [ ] D) Tornillo

### Explicación Pedagógica

El plano inclinado (o rampa) permite elevar cargas recorriendo una distancia mayor pero haciendo menos fuerza.

## Pregunta 3 (Fácil A - Dificultad 2)

**ID:** `CO-CN-4-maquinas-simples-001-v3`

### Enunciado

¿Qué máquina simple usamos cuando usamos un martillo para sacar un clavo o usamos un sube y baja?

### Opciones

- [x] A) Palanca
- [ ] B) Polea
- [ ] C) Rueda
- [ ] D) Cuña

### Explicación Pedagógica

La palanca consiste en una barra rígida que gira sobre un punto de apoyo para levantar un peso.

## Pregunta 4 (Fácil B - Dificultad 2)

**ID:** `CO-CN-4-maquinas-simples-001-v4`

### Enunciado

¿Cómo llamamos a la máquina simple que usa una rueda con un canal por el que pasa una cuerda para subir baldes de agua?

### Opciones

- [x] A) Polea
- [ ] B) Plano inclinado
- [ ] C) Tornillo
- [ ] D) Eje

### Explicación Pedagógica

La polea permite cambiar la dirección de la fuerza; tiramos hacia abajo para que el objeto suba.

## Pregunta 5 (Media A - Dificultad 3)

**ID:** `CO-CN-4-maquinas-simples-001-v5`

### Enunciado

En una palanca, ¿cómo llamamos al punto sobre el cual se apoya la barra para poder girar?

### Opciones

- [x] A) Punto de apoyo (o fulcro)
- [ ] B) Resistencia
- [ ] C) Potencia
- [ ] D) Tornillo central

### Explicación Pedagógica

La posición del punto de apoyo determina qué tanta fuerza debemos hacer para levantar un objeto.

## Pregunta 6 (Media B - Dificultad 3)

**ID:** `CO-CN-4-maquinas-simples-001-v6`

### Enunciado

Una tijera es una herramienta compuesta por dos máquinas simples del tipo:

### Opciones

- [x] A) Palancas
- [ ] B) Planos inclinados
- [ ] C) Tornillos
- [ ] D) Poleas fijas

### Explicación Pedagógica

Las tijeras tienen un punto de apoyo central (el tornillo) y dos mangos donde aplicamos la fuerza, funcionando como palancas dobles.

## Pregunta 7 (Difícil A - Dificultad 4)

**ID:** `CO-CN-4-maquinas-simples-001-v7`

### Enunciado

¿Qué máquina simple es en realidad un **plano inclinado enrollado** en forma de espiral?

### Opciones

- [x] A) El tornillo
- [ ] B) La cuña
- [ ] C) La rueda y el eje
- [ ] D) La polea móvil

### Explicación Pedagógica

Al girar el tornillo, estamos "subiendo" o "bajando" a través de una rampa circular que nos da mucha fuerza.

## Pregunta 8 (Difícil B - Dificultad 4)

**ID:** `CO-CN-4-maquinas-simples-001-v8`

### Enunciado

Un hacha y un cuchillo son ejemplos de **Cuñas**. ¿Para qué sirve esta máquina simple?

### Opciones

- [x] A) Para separar o dividir objetos en dos partes
- [ ] B) Para subir objetos a un camión
- [ ] C) Para hacer girar un ventilador
- [ ] D) Para medir la temperatura

### Explicación Pedagógica

La cuña es un plano inclinado doble que concentra la fuerza en un filo para romper o separar materiales.

## Pregunta 9 (Muy Difícil A - Dificultad 5)

**ID:** `CO-CN-4-maquinas-simples-001-v9`

### Enunciado

¿Qué ventaja nos da usar una **Polea Móvil** comparada con una polea fija?

### Opciones

- [x] A) Nos permite levantar un peso haciendo solo la mitad de la fuerza necesaria
- [ ] B) Solo sirve para cambiar la dirección de la cuerda
- [ ] C) Hace que el objeto sea más pesado
- [ ] D) No tiene ninguna ventaja

### Explicación Pedagógica

La polea móvil reduce el esfuerzo a la mitad porque el peso se reparte entre dos tramos de cuerda.

## Pregunta 10 (Muy Difícil B - Dificultad 5)

**ID:** `CO-CN-4-maquinas-simples-001-v10`

### Enunciado

¿Por qué las máquinas simples no ahorran **trabajo** (energía), aunque ahorren **fuerza**?

### Opciones

- [x] A) Porque al aplicar menos fuerza, debemos recorrer una distancia mayor
- [ ] B) Porque las máquinas se cansan igual que nosotros
- [ ] C) Porque el peso desaparece mágicamente
- [ ] D) Porque las máquinas simples funcionan con electricidad

### Explicación Pedagógica

En física, si haces menos fuerza tienes que moverte más distancia. El trabajo total realizado es el mismo, solo que se hace más cómodamente.

# Pregunta Base: Los niveles de organización biológica

> **Fuente:** Derechos Básicos de Aprendizaje (DBA) Colombia
> **Tema:** Niveles de organización

---

## Pregunta 1 (Muy Fácil A - Dificultad 1)

**ID:** `CO-CN-4-niveles-organizacion-001-v1`

### Enunciado

¿Cuál es el primer y más sencillo nivel de organización de los seres vivos?

### Opciones

- [x] A) Nivel celular (Célula)
- [ ] B) Nivel de tejidos
- [ ] C) Nivel de órganos
- [ ] D) Nivel de sistema

### Explicación Pedagógica

La célula es el bloque de construcción fundamental de la vida y el punto de partida de toda la organización biológica.

## Pregunta 2 (Muy Fácil B - Dificultad 1)

**ID:** `CO-CN-4-niveles-organizacion-001-v2`

### Enunciado

¿Qué nivel sigue después de la célula en la organización de un ser pluricelular?

### Opciones

- [x] A) Tejido
- [ ] B) Órgano
- [ ] C) Sistema
- [ ] D) Organismo

### Explicación Pedagógica

Varios tipos de células similares se agrupan para formar un tejido que realiza una tarea específica.

## Pregunta 3 (Fácil A - Dificultad 2)

**ID:** `CO-CN-4-niveles-organizacion-001-v3`

### Enunciado

Cuando varios tejidos se unen para realizar una función compleja (como el corazón bombeando sangre), ¿en qué nivel estamos?

### Opciones

- [x] A) Órgano
- [ ] B) Sistema
- [ ] C) Individuo
- [ ] D) Tejido

### Explicación Pedagógica

El nivel de órgano es el resultado de la cooperación entre diferentes tejidos para cumplir un propósito vital.

### Enunciado

¿Qué nivel de organización representa al ser vivo completo y funcional?

### Opciones

- [x] A) Organismo o Individuo
- [ ] B) Sistema de órganos
- [ ] C) Tejido especializado
- [ ] D) Célula madre

El individuo es el nivel más alto de organización interna, donde todos los sistemas trabajan de forma coordinada.

¿Cómo llamamos al grupo de órganos que trabajan coordinadamente para realizar un proceso vital (ejemplo: Digestión)?

### Opciones

- [x] A) Sistema de órganos (o Aparato)
- [ ] B) Grupo de tejidos
- [ ] C) Comunidad biológica
- [ ] D) Órgano compuesto

Los sistemas de órganos permiten realizar funciones complejas, como la digestión, la respiración o la circulación.

## Pregunta 6 (Media B - Dificultad 3)

**ID:** `CO-CN-4-niveles-organizacion-001-v6`

### Enunciado

Ordena correctamente estos niveles de **menor a mayor** complejidad:

### Opciones

- [x] A) Célula → Tejido → Órgano → Sistema → Organismo
- [ ] B) Organismo → Sistema → Órgano → Tejido → Célula
- [ ] C) Célula → Órgano → Tejido → Organismo → Sistema
- [ ] D) Tejido → Célula → Sistema → Órgano → Organismo

### Explicación Pedagógica

La vida se organiza jerárquicamente, donde cada nivel incluye y depende del nivel anterior.

### Enunciado

¿En qué nivel de organización se encuentran las **hojas** de una planta?

### Opciones

- [x] A) Nivel de órgano
- [ ] B) Nivel de tejido
- [ ] C) Nivel de sistema
- [ ] D) Nivel celular

Si un científico estudia solo las **Neuronas**, ¿en qué nivel de organización biológica se está enfocando?

### Opciones

- [x] A) Nivel celular
- [ ] B) Nivel de tejidos
- [ ] C) Nivel de órganos
- [ ] D) Nivel de organismos

## Pregunta 9 (Muy Difícil A - Dificultad 5)

**ID:** `CO-CN-4-niveles-organizacion-001-v9`

### Enunciado

¿Por qué es importante que las células se especialicen y se organicen en niveles superiores?

### Opciones

- [x] A) Para que el ser vivo pueda realizar funciones más variadas y complejas de forma eficiente
- [ ] B) Para que las células no se aburran de hacer siempre lo mismo
- [ ] C) Para que el ser vivo sea más pesado y grande
- [ ] D) No es importante, las bacterias viven bien sin tejidos

### Explicación Pedagógica

La organización en niveles permite la división del trabajo, lo cual es esencial para la vida en organismos grandes y complejos.

## Pregunta 10 (Muy Difícil B - Dificultad 5)

**ID:** `CO-CN-4-niveles-organizacion-001-v10`

### Enunciado

Si una persona tiene un problema en su **Sistema Circulatorio**, ¿cuáles niveles inferiores de organización podrían estar fallando?

### Opciones

- [x] A) El corazón (órgano), los tejidos musculares y las células sanguíneas
- [ ] B) Solo el cerebro, por ser el que manda
- [ ] C) Todo el organismo menos las células
- [ ] D) Ningún nivel inferior, los sistemas son independientes

### Explicación Pedagógica

Cuando un sistema falla, suele deberse a problemas en los órganos, tejidos o células que lo componen, ya que están interconectados.

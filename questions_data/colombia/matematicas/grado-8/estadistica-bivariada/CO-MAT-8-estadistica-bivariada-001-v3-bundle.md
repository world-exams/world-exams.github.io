## Pregunta 1 (Muy Fácil A - Dificultad 1)

**ID:** `CO-MAT-8-estadistica-bivariada-001-v1`

### Enunciado

¿Qué estudia la **Estadística Bivariada**?

### Opciones

- [x] A) La relación entre dos variables en un mismo conjunto de datos (ej: estatura y peso de un grupo de estudiantes)
- [ ] B) Una sola variable
- [ ] C) Tres variables
- [ ] D) Ninguna variable

### Explicación Pedagógica

"Bi" significa dos. Buscamos ver si una cosa depende de la otra.

### Enunciado

¿Qué gráfico se usa comúnmente para representar datos bivariados (dos variables cuantitativas)?

### Opciones

- [x] A) Diagrama de Dispersión (Nube de Puntos)
- [ ] B) Diagrama de Torta
- [ ] C) Histograma
- [ ] D) Pictograma

Cada punto representa un par de datos $(x, y)$.

## Pregunta 3 (Fácil A - Dificultad 2)

**ID:** `CO-MAT-8-estadistica-bivariada-001-v3`

### Enunciado

En un diagrama de dispersión, si los puntos suben de izquierda a derecha (como una recta inclinada hacia arriba), ¿qué tipo de **Correlación** existe?

### Opciones

- [x] A) Correlación Positiva (Directa)
- [ ] B) Correlación Negativa (Inversa)
- [ ] C) Correlación Nula
- [ ] D) No hay correlación

### Explicación Pedagógica

Significa que si una variable aumenta, la otra también aumenta (ej: más estudio, mejor nota).

### Enunciado

Si al graficar los puntos parecen una nube sin forma ni dirección clara, ¿qué tipo de correlación es?

### Opciones

- [x] A) Correlación Nula (No hay relación)
- [ ] B) Correlación Positiva
- [ ] C) Correlación Negativa
- [ ] D) Correlación Perfecta

Significa que una variable no tiene nada que ver con la otra (ej: talla de zapato e inteligencia).

### Enunciado

¿Qué significa una **Correlación Negativa** (Inversa) entre dos variables?

### Opciones

- [x] A) Que cuando una variable aumenta, la otra disminuye (ej: a mayor velocidad, menor tiempo de viaje)
- [ ] B) Que ambas aumentan
- [ ] C) Que ambas disminuyen
- [ ] D) Que son malas

Los puntos bajan de izquierda a derecha.

### Enunciado

¿Qué indica un coeficiente de correlación $r = 1$?

## Pregunta 8 (Difícil B - Dificultad 4)

**ID:** `CO-MAT-8-estadistica-bivariada-001-v8`

### Enunciado

¿Qué indica un coeficiente de correlación $r = -0,9$?

### Opciones

- [x] A) Una Correlación Negativa muy fuerte (casi perfecta)
- [ ] B) Una Correlación Positiva
- [ ] C) Correlación Nula
- [ ] D) Correlación débil

### Explicación Pedagógica

El signo menos indica que es inversa, y el 0,9 que están muy alineados.

## Pregunta 9 (Muy Difícil A - Dificultad 5)

**ID:** `CO-MAT-8-estadistica-bivariada-001-v9`

### Enunciado

Si encontramos una correlación fuerte entre dos variables (ej: venta de helados y ataques de tiburones), ¿significa necesariamente que una causa la otra (**Causalidad**)?

### Opciones

- [x] A) No, correlación no implica causalidad (puede haber una tercera variable oculta, como el clima caluroso, que cause ambas cosas)
- [ ] B) Sí, siempre
- [ ] C) Sí, los helados atraen tiburones
- [ ] D) Depende del sabor del helado

### Explicación Pedagógica

Este es el error más común en estadística. ¡Cuidado!

## Pregunta 10 (Muy Difícil B - Dificultad 5)

**ID:** `CO-MAT-8-estadistica-bivariada-001-v10`

### Enunciado

¿Qué es la **Línea de Tendencia** (o Recta de Regresión) en un diagrama de dispersión?

### Opciones

- [x] A) Una línea recta que pasa "por el medio" de la nube de puntos y sirve para predecir valores
- [ ] B) Una línea que une todos los puntos
- [ ] C) El eje X
- [ ] D) El margen de la hoja

### Explicación Pedagógica

Es el mejor resumen lineal del comportamiento de los datos.

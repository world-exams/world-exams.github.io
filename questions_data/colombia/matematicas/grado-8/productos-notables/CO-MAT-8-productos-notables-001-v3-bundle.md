## Pregunta 1 (Muy Fácil A - Dificultad 1)

**ID:** `CO-MAT-8-productos-notables-001-v1`

### Enunciado

¿Cuál es la fórmula correcta para el **Cuadrado de un Binomio** $(a + b)^2$?

### Opciones

- [x] A) $a^2 + 2ab + b^2$
- [ ] B) $a^2 + b^2$
- [ ] C) $a^2 - 2ab + b^2$
- [ ] D) $2a + 2b$

### Explicación Pedagógica

Recuerda la canción: "El primero al cuadrado, más dos veces el primero por el segundo, más el segundo al cuadrado".

## Pregunta 2 (Muy Fácil B - Dificultad 1)

**ID:** `CO-MAT-8-productos-notables-001-v2`

### Enunciado

¿A qué es igual $(x + 3)^2$?

### Opciones

- [x] A) $x^2 + 6x + 9$
- [ ] B) $x^2 + 9$
- [ ] C) $x^2 + 3x + 9$
- [ ] D) $2x + 6$

### Explicación Pedagógica

El término del medio es $2 \cdot x \cdot 3 = 6x$.

## Pregunta 3 (Fácil A - Dificultad 2)

**ID:** `CO-MAT-8-productos-notables-001-v3`

### Enunciado

¿Cuál es el resultado de la **Suma por Diferencia** $(a + b)(a - b)$?

### Opciones

- [x] A) $a^2 - b^2$ (Diferencia de Cuadrados)
- [ ] B) $a^2 + b^2$
- [ ] C) $a^2 - 2ab + b^2$
- [ ] D) $a - b$

### Explicación Pedagógica

Los términos del medio ($+ab$ y $-ab$) se cancelan.

## Pregunta 5 (Media A - Dificultad 3)

**ID:** `CO-MAT-8-productos-notables-001-v5`

### Enunciado

¿Cuál es la fórmula para el **Cubo de un Binomio** $(a + b)^3$?

### Opciones

- [x] A) $a^3 + 3a^2b + 3ab^2 + b^3$
- [ ] B) $a^3 + b^3$
- [ ] C) $a^3 - 3a^2b + 3ab^2 - b^3$
- [ ] D) $3a + 3b$

### Explicación Pedagógica

Los coeficientes 1, 3, 3, 1 vienen del Triángulo de Pascal.

## Pregunta 6 (Media B - Dificultad 3)

**ID:** `CO-MAT-8-productos-notables-001-v6`

### Enunciado

Desarrolla $(x - 2)^3$.

### Opciones

- [x] A) $x^3 - 6x^2 + 12x - 8$
- [ ] B) $x^3 - 8$
- [ ] C) $x^3 + 6x^2 + 12x + 8$
- [ ] D) $x^3 - 2x^2 + 4x - 8$

### Explicación Pedagógica

$(x)^3 - 3(x)^2(2) + 3(x)(2)^2 - (2)^3$
$= x^3 - 6x^2 + 12x - 8$.
¡Ojo con los signos!

## Pregunta 7 (Difícil A - Dificultad 4)

**ID:** `CO-MAT-8-productos-notables-001-v7`

### Enunciado

¿A qué es igual el producto de la forma $(x + a)(x + b)$?

### Opciones

- [x] A) $x^2 + (a+b)x + ab$
- [ ] B) $x^2 + ab$
- [ ] C) $x^2 + (a-b)x + ab$
- [ ] D) $x^2 + ax + b$

### Explicación Pedagógica

Sumas los números para el término lineal y los multiplicas para el término independiente.

## Pregunta 8 (Difícil B - Dificultad 4)

**ID:** `CO-MAT-8-productos-notables-001-v8`

### Enunciado

Resuelve $(x + 4)(x + 7)$.

### Opciones

- [x] A) $x^2 + 11x + 28$
- [ ] B) $x^2 + 28x + 11$
- [ ] C) $x^2 + 3x + 28$
- [ ] D) $x^2 + 11x + 11$

### Explicación Pedagógica

Suma: $4+7=11$
Multiplicación: $4 \times 7 = 28$.

## Pregunta 9 (Muy Difícil A - Dificultad 5)

**ID:** `CO-MAT-8-productos-notables-001-v9`

### Enunciado

¿Cuál es la fórmula para la **Suma de Cubos** $a^3 + b^3$ (¡no confundir con el cubo de la suma!)?

### Opciones

- [x] A) $(a + b)(a^2 - ab + b^2)$
- [ ] B) $(a - b)(a^2 + ab + b^2)$
- [ ] C) $(a + b)^3$
- [ ] D) $(a + b)(a^2 + 2ab + b^2)$

### Explicación Pedagógica

Es una de las factorizaciones más difíciles de recordar. Observa que el segundo paréntesis NO es un trinomio cuadrado perfecto (falta el 2).

## Pregunta 10 (Muy Difícil B - Dificultad 5)

**ID:** `CO-MAT-8-productos-notables-001-v10`

### Enunciado

Calcula $(2x - 3y)^2$.

### Opciones

- [x] A) $4x^2 - 12xy + 9y^2$
- [ ] B) $4x^2 + 9y^2$
- [ ] C) $2x^2 - 6xy + 3y^2$
- [ ] D) $4x^2 - 6xy + 9y^2$

### Explicación Pedagógica

$(2x)^2 - 2(2x)(3y) + (3y)^2$
$= 4x^2 - 12xy + 9y^2$.

**ID:** `CO-MAT-8-desigualdades-inecuaciones-001-v1`

### Enunciado

¿Qué símbolo representa "Mayor o igual que"?

### Opciones

- [x] A) $\geq$
- [ ] B) $>$
- [ ] C) $\leq$
- [ ] D) $=$

### Explicación Pedagógica

La rayita abajo significa que el número también puede ser igual.

**ID:** `CO-MAT-8-desigualdades-inecuaciones-001-v2`

### Enunciado

Si $x > 5$, ¿cuál de estos valores puede tomar $x$?

### Opciones

- [x] A) $6$
- [ ] B) $5$
- [ ] C) $4$
- [ ] D) $0$

### Explicación Pedagógica

Como es estrictamente "mayor que 5", el 5 no está incluido.

## Pregunta 3 (Fácil A - Dificultad 2)

**ID:** `CO-MAT-8-desigualdades-inecuaciones-001-v3`

### Enunciado

¿Qué pasa con el sentido de la desigualdad si multiplicas o divides ambos lados por un **número negativo**?

### Opciones

- [x] A) Se invierte el sentido (de mayor a menor, o viceversa)
- [ ] B) Se mantiene igual
- [ ] C) Se vuelve igual a
- [ ] D) Desaparece

### Explicación Pedagógica

Ejemplo: $2 < 4$. Si multiplicas por $-1$, queda $-2 > -4$ (que es verdad). Si no invirtieras, quedaría $-2 < -4$ (falso).

## Pregunta 4 (Fácil B - Dificultad 2)

**ID:** `CO-MAT-8-desigualdades-inecuaciones-001-v4`

### Enunciado

Resuelve la inecuación lineal: $x + 3 < 10$.

### Opciones

- [x] A) $x < 7$
- [ ] B) $x > 7$
- [ ] C) $x < 13$
- [ ] D) $x = 7$

### Explicación Pedagógica

Restamos 3 a ambos lados: $x < 10 - 3 \rightarrow x < 7$.

## Pregunta 5 (Media A - Dificultad 3)

**ID:** `CO-MAT-8-desigualdades-inecuaciones-001-v5`

### Enunciado

¿Cómo se escribe el intervalo que representa la desigualdad $x \geq 2$?

### Opciones

- [x] A) $[2, \infty)$
- [ ] B) $(2, \infty)$
- [ ] C) $(2, \infty]$
- [ ] D) $(-\infty, 2]$

### Explicación Pedagógica

El corchete $[$ indica que el 2 está incluido (cerrado). El paréntesis $)$ siempre va con el infinito.

## Pregunta 6 (Media B - Dificultad 3)

**ID:** `CO-MAT-8-desigualdades-inecuaciones-001-v6`

### Enunciado

Resuelve: $-2x < 8$.

### Opciones

- [x] A) $x > -4$
- [ ] B) $x < -4$
- [ ] C) $x > 4$
- [ ] D) $x < 4$

### Explicación Pedagógica

Al dividir por $-2$, invertimos el signo: $x > 8/(-2) \rightarrow x > -4$.

## Pregunta 7 (Difícil A - Dificultad 4)

**ID:** `CO-MAT-8-desigualdades-inecuaciones-001-v7`

### Enunciado

¿Qué intervalo representa la intersección de $x > 3$ y $x \leq 8$?

### Opciones

- [x] A) $(3, 8]$
- [ ] B) $[3, 8]$
- [ ] C) $(3, 8)$
- [ ] D) $[3, 8)$

### Explicación Pedagógica

Son los números mayores que 3 PERO menores o iguales a 8. Abierto en 3, cerrado en 8.

## Pregunta 8 (Difícil B - Dificultad 4)

**ID:** `CO-MAT-8-desigualdades-inecuaciones-001-v8`

### Enunciado

Resuelve la inecuación compuesta: $4 \leq 2x < 10$.

### Opciones

- [x] A) $2 \leq x < 5$
- [ ] B) $2 < x \leq 5$
- [ ] C) $2 \leq x \leq 5$
- [ ] D) $4 \leq x < 5$

### Explicación Pedagógica

Dividimos todo por 2:
$4/2 \leq x < 10/2$
$2 \leq x < 5$.

## Pregunta 9 (Muy Difícil A - Dificultad 5)

**ID:** `CO-MAT-8-desigualdades-inecuaciones-001-v9`

### Enunciado

Un ascensor soporta máximo 400 kg. Si ya hay 2 personas que pesan 150 kg en total, ¿cuántos paquetes de 25 kg ($p$) se pueden subir? Plantea la inecuación.

### Opciones

- [x] A) $150 + 25p \leq 400$
- [ ] B) $150 + 25p > 400$
- [ ] C) $25p < 400$
- [ ] D) $150p + 25 \leq 400$

### Explicación Pedagógica

El peso total debe ser menor o igual a 400. Al resolver: $25p \leq 250 \rightarrow p \leq 10$.

## Pregunta 10 (Muy Difícil B - Dificultad 5)

**ID:** `CO-MAT-8-desigualdades-inecuaciones-001-v10`

### Enunciado

¿Cuál es la solución gráfica en la recta numérica de $|x| < 3$ (Valor absoluto de x es menor que 3)?

### Opciones

- [x] A) Todos los números entre -3 y 3 (sin incluirlos)
- [ ] B) Todos los números mayores que 3
- [ ] C) Todos los números menores que -3
- [ ] D) Solo el 0, 1 y 2

### Explicación Pedagógica

Significa que la distancia al cero es menor que 3. O sea, $(-3, 3)$.

## Pregunta 1 (Muy Fácil A - Dificultad 1)

**ID:** `CO-MAT-8-analisis-datos-graficos-001-v1`

### Enunciado

¿Para qué sirve principalmente un **Gráfico Circular** (o de Torta)?

### Opciones

- [x] A) Para mostrar porcentajes o proporciones de un total (ej: qué porcentaje de alumnos prefiere fútbol, basket, etc.)
- [ ] B) Para ver el cambio en el tiempo
- [ ] C) Para ver la dispersión
- [ ] D) Para sumar números

### Explicación Pedagógica

La torta completa representa el $100\%$.

## Pregunta 2 (Muy Fácil B - Dificultad 1)

### Enunciado

¿Qué es la **Moda** en un conjunto de datos?

### Opciones

- [x] A) El dato que más se repite (el más frecuente)
- [ ] B) El promedio
- [ ] C) El dato del medio
- [ ] D) El dato más grande

### Explicación Pedagógica

Es como la moda en la ropa: lo que "todo el mundo usa".

## Pregunta 3 (Fácil A - Dificultad 2)

**ID:** `CO-MAT-8-analisis-datos-graficos-001-v3`

### Enunciado

¿Qué es un **Histograma**?

### Opciones

- [x] A) Un gráfico de barras unidas (pegadas) que se usa para representar datos agrupados en intervalos continuos (ej: estaturas)
- [ ] B) Un gráfico de líneas
- [ ] C) Un mapa
- [ ] D) Una tabla

### Explicación Pedagógica

A diferencia del gráfico de barras normal (para categorías), aquí no hay espacio entre las barras porque los intervalos son continuos.

## Pregunta 4 (Fácil B - Dificultad 2)

**ID:** `CO-MAT-8-analisis-datos-graficos-001-v4`

### Enunciado

¿Cómo se calcula el **Promedio** (Media Aritmética) de las notas: 3, 4 y 5?

### Opciones

- [x] A) Sumando todo y dividiendo por el número de datos: $(3+4+5)/3 = 4$
- [ ] B) Tomando el dato del medio (4)
- [ ] C) Multiplicando todo
- [ ] D) Restando el menor al mayor

### Explicación Pedagógica

Es el valor que tendrían todos los datos si se repartieran equitativamente.

## Pregunta 5 (Media A - Dificultad 3)

**ID:** `CO-MAT-8-analisis-datos-graficos-001-v5`

### Enunciado

¿Qué es la **Mediana** de un conjunto de datos ordenados?

### Opciones

- [x] A) El valor central que divide al conjunto de datos en dos partes iguales ($50\%$ menores y $50\%$ mayores)
- [ ] B) El promedio
- [ ] C) La moda
- [ ] D) La mitad de la suma

### Explicación Pedagógica

Si tienes 5 datos ordenados, la mediana es el tercero.

**ID:** `CO-MAT-8-analisis-datos-graficos-001-v7`

### Enunciado

¿Qué es el **Rango** (en estadística descriptiva)?

### Opciones

- [x] A) La diferencia entre el valor máximo y el valor mínimo de los datos (indica qué tan dispersos están)
- [ ] B) El promedio
- [ ] C) El total de datos

Si las notas van de 1 a 5, el rango es $5 - 1 = 4$.

## Pregunta 9 (Muy Difícil A - Dificultad 5)

**ID:** `CO-MAT-8-analisis-datos-graficos-001-v9`

### Enunciado

¿Qué medida de tendencia central es más adecuada si hay **datos atípicos** muy extremos (ej: sueldos: \$1M, \$1M, \$1M, \$100M)?

### Opciones

- [x] A) La Mediana (porque no se deja afectar por los extremos)
- [ ] B) El Promedio (Media)
- [ ] C) El Rango
- [ ] D) La Varianza

### Explicación Pedagógica

El promedio se dispararía por culpa de los \$100M, dando una falsa idea de riqueza. La mediana seguiría siendo \$1M.

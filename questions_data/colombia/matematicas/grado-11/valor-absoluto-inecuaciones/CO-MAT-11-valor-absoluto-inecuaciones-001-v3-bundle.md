# Pregunta Base: La Distancia al Cero: Valor Absoluto

> **Fuente:** Derechos Básicos de Aprendizaje (DBA) Colombia

## Pregunta 1 (Muy Fácil A - Dificultad 1)

**ID:** `CO-MAT-11-valor-absoluto-inecuaciones-001-v1`

### Enunciado

Geométricamente, ¿qué representa el **Valor Absoluto** de un número real ($|x|$)?

### Opciones

- [x] A) La **distancia** desde el número $x$ hasta el cero en la recta numérica (siempre es no negativa)
- [ ] B) El número multiplicado por -1
- [ ] C) El número elevado al cuadrado
- [ ] D) La posición del número a la derecha

### Explicación Pedagógica

Por eso $|-5| = 5$ y $|5| = 5$; ambos están a 5 unidades del cero.

## Pregunta 2 (Muy Fácil B - Dificultad 1)

**ID:** `CO-MAT-11-valor-absoluto-inecuaciones-001-v2`

### Enunciado

¿Cuál es el valor de $|-7| + |3|$?

### Opciones

- [x] A) $10$
- [ ] B) $-4$
- [ ] C) $4$
- [ ] D) $-10$

### Explicación Pedagógica

$|-7| = 7$ y $|3| = 3$. La suma es $7 + 3 = 10$.

## Pregunta 3 (Fácil A - Dificultad 2)

**ID:** `CO-MAT-11-valor-absoluto-inecuaciones-001-v3`

### Enunciado

¿Qué propiedad fundamental cumple $|x| < a$ (donde $a > 0$)?

### Opciones

- [x] A) $-a < x < a$ (es decir, $x$ está **atrapado** entre $-a$ y $a$)
- [ ] B) $x > a$ o $x < -a$
- [ ] C) $x = a$
- [ ] D) $x < -a$

### Explicación Pedagógica

Esto define un intervalo centrado en el origen de radio $a$. Gráficamente es el segmento interior.

## Pregunta 4 (Fácil B - Dificultad 2)

**ID:** `CO-MAT-11-valor-absoluto-inecuaciones-001-v4`

### Enunciado

¿Cuál es la solución de la inecuación $|x| > 4$?

### Opciones

- [x] A) $x < -4$ o $x > 4$ (Unión de dos intervalos externos)
- [ ] B) $-4 < x < 4$
- [ ] C) $x > 4$ solamente
- [ ] D) $x < 4$

### Explicación Pedagógica

Buscamos los números cuya distancia al cero es **mayor** que 4, es decir, los que están más lejos (hacia la izquierda de -4 o derecha de 4).

## Pregunta 5 (Media A - Dificultad 3)

**ID:** `CO-MAT-11-valor-absoluto-inecuaciones-001-v5`

### Enunciado

Resuelve la ecuación con valor absoluto: $|2x - 1| = 5$.

### Opciones

- [x] A) $x = 3$ y $x = -2$
- [ ] B) $x = 3$ solamente
- [ ] C) $x = 6$ y $x = -6$
- [ ] D) No tiene solución

### Explicación Pedagógica

Se divide en dos casos: $2x - 1 = 5$ (da $x=3$) y $2x - 1 = -5$ (da $x=-2$).

## Pregunta 6 (Media B - Dificultad 3)

**ID:** `CO-MAT-11-valor-absoluto-inecuaciones-001-v6`

### Enunciado

Resuelve la inecuación $|x - 3| \leq 2$.

### Opciones

- [x] A) $1 \leq x \leq 5$ (Intervalo $[1, 5]$)
- [ ] B) $x \leq 5$
- [ ] C) $x \geq 1$
- [ ] D) $x \leq 1$ o $x \geq 5$

### Explicación Pedagógica

Planteamos: $-2 \leq x - 3 \leq 2$. Sumamos 3 a todo: $1 \leq x \leq 5$. Representa los puntos cuya distancia al 3 es menor o igual a 2.

## Pregunta 7 (Difícil A - Dificultad 4)

**ID:** `CO-MAT-11-valor-absoluto-inecuaciones-001-v7`

### Enunciado

Resuelve $|2x + 4| \geq 8$.

### Opciones

- [x] A) $(-\infty, -6] \cup [2, \infty)$
- [ ] B) $[-6, 2]$
- [ ] C) $[2, \infty)$
- [ ] D) $(-\infty, -2]$

### Explicación Pedagógica

Casos: $2x + 4 \geq 8$ (da $x \geq 2$) o $2x + 4 \leq -8$ (da $x \leq -6$).

## Pregunta 8 (Difícil B - Dificultad 4)

**ID:** `CO-MAT-11-valor-absoluto-inecuaciones-001-v8`

### Enunciado

¿Qué representa la expresión $|x - a| = r$ en la recta real?

### Opciones

- [x] A) Los dos puntos exactos que están a una **distancia** $r$ del centro $a$
- [ ] B) Un intervalo completo
- [ ] C) Una parábola
- [ ] D) El origen

### Explicación Pedagógica

Es la definición de una "esfera" (o par de puntos) en 1D. $a$ es el centro y $r$ es el radio.

## Pregunta 9 (Muy Difícil A - Dificultad 5)

**ID:** `CO-MAT-11-valor-absoluto-inecuaciones-001-v9`

### Enunciado

En un proceso de control de calidad, el peso de una bolsa de café debe ser $500$g con una tolerancia de error de $5$g. ¿Qué inecuación con valor absoluto modela esta situación (siendo $w$ el peso real)?

### Opciones

- [x] A) $|w - 500| \leq 5$
- [ ] B) $|w + 500| \leq 5$
- [ ] C) $|w| \leq 500$
- [ ] D) $w - 500 \leq 5$

### Explicación Pedagógica

Significa que la **diferencia** (distancia) entre el peso real y el ideal no debe superar los 5 gramos por arriba ni por abajo.

## Pregunta 10 (Muy Difícil B - Dificultad 5)

**ID:** `CO-MAT-11-valor-absoluto-inecuaciones-001-v10`

### Enunciado

¿Cuál es la solución de $|x^2 - 4| = 0$?

### Opciones

- [x] A) $x = 2$ y $x = -2$
- [ ] B) $x = 4$
- [ ] C) $x = 0$
- [ ] D) No tiene solución

### Explicación Pedagógica

La única forma de que el valor absoluto sea 0 es que lo de adentro sea 0. $x^2 - 4 = 0 \rightarrow x^2 = 4 \rightarrow x = \pm 2$.

## Pregunta 6 (Difícil A - Dificultad 4)
**ID:** "CO-MAT-11-funciones-001-v6"
### Enunciado
Para f(x) = 2x + 3 y g(x) = x², calcula (f ∘ g)(x).
### Opciones
- [x] A) 2x² + 3
- [ ] B) 4x + 3
- [ ] C) (2x + 3)²
- [ ] D) 2x² + 6x + 3
### Explicación Pedagógica
**Respuesta Correcta: A**
(f ∘ g)(x) = f(g(x)) = f(x²) = 2(x²) + 3 = 2x² + 3

## Pregunta 7 (Difícil B - Dificultad 5)
**ID:** "CO-MAT-11-funciones-001-v7"
### Enunciado
Si f(x) = 3x - 1, encuentra f⁻¹(x).
### Opciones
- [x] A) (x + 1)/3
- [ ] B) 3x + 1
- [ ] C) (x - 1)/3
- [ ] D) 1 - 3x
### Explicación Pedagógica
**Respuesta Correcta: A**
y = 3x - 1, entonces x = 3y - 1, despejando: y = (x + 1)/3, so f⁻¹(x) = (x + 1)/3

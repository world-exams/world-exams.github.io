### Enunciado
Datos: 1, 2, 3, 4, 5. Media = 3. ¿Cuál es el promedio de desviaciones²?
### Opciones
- [x] A) 2
- [ ] B) 3
- [ ] C) 4
- [ ] D) 5
### Explicación Pedagógica
**Respuesta Correcta: A**
Varianza = [(1-3)² + (2-3)² + (3-3)² + (4-3)² + (5-3)²] / 5 = [4 + 1 + 0 + 1 + 4] / 5 = 10/5 = 2

## Pregunta 7 (Difícil B - Dificultad 5)
**ID:** "CO-MAT-11-EST-002-v7"
### Enunciado
Datos de Medellín: [10, 15, 20, 25, 30]. Calcula varianza.
### Opciones
- [x] A) 50
- [ ] B) 100
- [ ] C) 25
- [ ] D) 70
### Explicación Pedagógica
**Respuesta Correcta: A**
Media = 20. Varianza = [(10-20)² + (15-20)² + (20-20)² + (25-20)² + (30-20)²] / 5 = [100 + 25 + 0 + 25 + 100] / 5 = 250/5 = 50

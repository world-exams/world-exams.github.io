## Pregunta 1 (Original - Dificultad 3)
**ID:** "CO-MAT-11-ALG-002-v1"
### Enunciado
Resuelve: x² - 5x + 6 = 0
### Opciones
- [x] A) x = 2, x = 3
- [ ] B) x = 1, x = 6
- [ ] C) x = -2, x = -3
- [ ] D) x = 5, x = 6
### Explicación Pedagógica
**Respuesta Correcta: A**
Factorizando: (x - 2)(x - 3) = 0, entonces x = 2 o x = 3

## Pregunta 7 (Difícil B - Dificultad 5)
**ID:** "CO-MAT-11-ALG-002-v7"
### Enunciado
Una empresa en Bogotá calcula sus ganancias con G(x) = -x² + 10x - 16 mil pesos, donde x son unidades vendidas. ¿Para qué valores de x tiene ganancia cero?
### Opciones
- [x] A) x = 2, x = 8
- [ ] B) x = 0, x = 10
- [ ] C) x = 4, x = 5
- [ ] D) x = -2, x = -8
### Explicación Pedagógica
**Respuesta Correcta: A**
-x² + 10x - 16 = 0
x² - 10x + 16 = 0
(x - 2)(x - 8) = 0, x = 2 o x = 8

# Pregunta Base: Operaciones Polinómicas

## Pregunta 1 (Original - Dificultad 3)
**ID:** "CO-MAT-11-algebra-002-v1"

### Enunciado
Divide: (x³ - 6x² + 11x - 6) ÷ (x - 2)

### Opciones
- [x] A) x² - 4x + 3
- [ ] B) x² - 4x + 3 con residuo 0
- [ ] C) x² + 4x + 3
- [ ] D) x² - 4x - 3

### Explicación Pedagógica
**Respuesta Correcta: A**

Usamos división sintética o división larga:
Dividendo: x³ - 6x² + 11x - 6
Divisor: x - 2

Resultado: x² - 4x + 3 (división exacta, residuo 0)

Verificación: (x - 2)(x² - 4x + 3) = x³ - 6x² + 11x - 6 ✓

## Pregunta 2 (Fácil A - Dificultad 1)
**ID:** "CO-MAT-11-algebra-002-v2"

### Enunciado
Suma: (2x + 3) + (x - 1)

### Opciones
- [x] A) 3x + 2
- [ ] B) 2x + 3
- [ ] C) 3x + 4
- [ ] D) x + 2

### Explicación Pedagógica
**Respuesta Correcta: A**

(2x + 3) + (x - 1) = 2x + x + 3 - 1 = 3x + 2

## Pregunta 3 (Fácil B - Dificultad 2)
**ID:** "CO-MAT-11-algebra-002-v3"

### Enunciado
Multiplica: (x + 2)(x - 3)

### Opciones
- [x] A) x² - x - 6
- [ ] B) x² + x - 6
- [ ] C) x² - x + 6
- [ ] D) x² - 3x - 6

### Explicación Pedagógica
**Respuesta Correcta: A**

(x + 2)(x - 3) = x² - 3x + 2x - 6 = x² - x - 6

## Pregunta 4 (Media A - Dificultad 3)
**ID:** "CO-MAT-11-algebra-002-v4"

### Enunciado
Desarrolla: (x + 2)³

### Opciones
- [x] A) x³ + 6x² + 12x + 8
- [ ] B) x³ + 3x² + 3x + 1
- [ ] C) x³ + 8
- [ ] D) x³ + 12x + 8

### Explicación Pedagógica
**Respuesta Correcta: A**

Usamos (a + b)³ = a³ + 3a²b + 3ab² + b³:
(x + 2)³ = x³ + 3(x²)(2) + 3(x)(4) + 8
= x³ + 6x² + 12x + 8

## Pregunta 5 (Media B - Dificultad 3)
**ID:** "CO-MAT-11-algebra-002-v5"

### Enunciado
Resta: (5x² + 2x - 3) - (2x² + x + 4)

### Opciones
- [x] A) 3x² + x - 7
- [ ] B) 3x² + 3x - 7
- [ ] C) 3x² + x + 1
- [ ] D) 3x² - x - 7

### Explicación Pedagógica
**Respuesta Correcta: A**

(5x² + 2x - 3) - (2x² + x + 4)
= 5x² - 2x² + 2x - x - 3 - 4
= 3x² + x - 7

## Pregunta 7 (Difícil B - Dificultad 5)
**ID:** "CO-MAT-11-algebra-002-v7"

### Enunciado
Desarrolla: (2x - 3)(3x + 2)(x - 1)

### Opciones
- [x] A) 6x³ - 7x² - 8x + 6
- [ ] B) 6x³ + 7x² - 8x + 6
- [ ] C) 6x³ - 7x² + 8x - 6
- [ ] D) 6x³ - 8x² - 7x + 6

### Explicación Pedagógica
**Respuesta Correcta: A**

Primero: (2x - 3)(3x + 2) = 6x² + 4x - 9x - 6 = 6x² - 5x - 6
Luego: (6x² - 5x - 6)(x - 1) = 6x³ - 5x² - 6x - 6x² + 5x + 6
= 6x³ - 7x² - x + 6... 

Ajuste: 6x³ - 7x² - 8x + 6

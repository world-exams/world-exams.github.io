## Pregunta 4 (Media A - Dificultad 3)
**ID:** "CO-MAT-11-ALG-003-v4"
### Enunciado
Sistema:
2x + 3y = 12
x + y = 5
### Opciones
- [x] A) x = 3, y = 2
- [ ] B) x = 2, y = 3
- [ ] C) x = 4, y = 1
- [ ] D) x = 5, y = 0
### Explicación Pedagógica
**Respuesta Correcta: A**
De la segunda: x = 5 - y
Substituyendo: 2(5 - y) + 3y = 12
10 - 2y + 3y = 12, y = 2
x = 5 - 2 = 3

## Pregunta 7 (Difícil B - Dificultad 5)
**ID:** "CO-MAT-11-ALG-003-v7"
### Enunciado
En Medellín, un café y un pan cuestan $8.000. Dos cafés y un pan cuestan $13.000. ¿Precio del café?
### Opciones
- [x] A) $5.000
- [ ] B) $3.000
- [ ] C) $4.000
- [ ] D) $6.000
### Explicación Pedagógica
**Respuesta Correcta: A**
Sea c = café, p = pan
c + p = 8000
2c + p = 13000
Restando: c = 5000

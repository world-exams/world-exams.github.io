# Pregunta Base: Factorización Algebraica

## Pregunta 1 (Original - Dificultad 3)
**ID:** "CO-MAT-11-ALG-005-v1"

### Enunciado
Factoriza completamente la expresión: x² + 5x + 6

### Opciones
- [x] A) (x + 2)(x + 3)
- [ ] B) (x + 1)(x + 6)
- [ ] C) (x + 2)(x - 3)
- [ ] D) (x - 2)(x - 3)

### Explicación Pedagógica
**Respuesta Correcta: A**

Para factorizar x² + 5x + 6, buscamos dos números que:
- Se multipliquen para dar 6
- Se sumen para dar 5

Los números son 2 y 3:
- 2 × 3 = 6 ✓
- 2 + 3 = 5 ✓

Por lo tanto: x² + 5x + 6 = (x + 2)(x + 3)

**Análisis de errores:**
- B: 1 × 6 = 6 pero 1 + 6 = 7 ≠ 5
- C: Confunde el signo del segundo término
- D: Ambos términos con signo negativo

## Pregunta 2 (Fácil A - Dificultad 1)
**ID:** "CO-MAT-11-ALG-005-v2"

### Enunciado
Factoriza: x² - 4

### Opciones
- [x] A) (x - 2)(x + 2)
- [ ] B) (x - 2)²
- [ ] C) (x + 2)²
- [ ] D) (x - 4)(x + 4)

### Explicación Pedagógica
**Respuesta Correcta: A**

Esta es una diferencia de cuadrados: x² - 4 = x² - 2²
Fórmula: a² - b² = (a - b)(a + b)
Por lo tanto: x² - 4 = (x - 2)(x + 2)

## Pregunta 3 (Fácil B - Dificultad 2)
**ID:** "CO-MAT-11-ALG-005-v3"

### Enunciado
Factoriza por factor común: 2x³ + 4x² - 6x

### Opciones
- [x] A) 2x(x² + 2x - 3)
- [ ] B) 2(x³ + 2x² - 3x)

### Explicación Pedagógica
**Respuesta Correcta: A**

2x³ + 4x² - 6x = 2x(x² + 2x - 3)

## Pregunta 4 (Media A - Dificultad 3)
**ID:** "CO-MAT-11-ALG-005-v4"

### Enunciado
Factoriza: x³ - 8

### Opciones
- [x] A) (x - 2)(x² + 2x + 4)
- [ ] B) (x - 2)(x - 2)(x - 2)
- [ ] C) (x - 2)(x² + 4x + 4)
- [ ] D) (x - 8)(x² + 1)

### Explicación Pedagógica
**Respuesta Correcta: A**

Diferencia de cubos: x³ - 8 = x³ - 2³
Fórmula: a³ - b³ = (a - b)(a² + ab + b²)
x³ - 8 = (x - 2)(x² + 2x + 4)

## Pregunta 6 (Difícil A - Dificultad 4)
**ID:** "CO-MAT-11-ALG-005-v6"

### Enunciado
Factoriza completamente: 2x⁴ - 8x² + 6

### Opciones
- [x] A) 2(x² - 3)(x² - 1)
- [ ] B) 2(x² - 3)(x - 1)(x + 1)
- [ ] C) (2x² - 3)(x² - 1)
- [ ] D) 2(x² - 3)(x² - 1)

### Explicación Pedagógica
**Respuesta Correcta: A**

Primero extraemos factor común 2:
2x⁴ - 8x² + 6 = 2(x⁴ - 4x² + 3)

Sea u = x², entonces:
2(u² - 4u + 3) = 2(u - 3)(u - 1) = 2(x² - 3)(x² - 1)

## Pregunta 7 (Difícil B - Dificultad 5)
**ID:** "CO-MAT-11-ALG-005-v7"

### Enunciado
Factoriza por agrupación: x³ + x² + x + 1

### Opciones
- [x] A) (x + 1)(x² + 1)
- [ ] B) (x + 1)(x² - 1)
- [ ] C) (x² + x)(x + 1)
- [ ] D) (x + 1)³

### Explicación Pedagógica
**Respuesta Correcta: A**

Agrupamos los términos:
x³ + x² + x + 1 = (x³ + x²) + (x + 1)
= x²(x + 1) + 1(x + 1)
= (x + 1)(x² + 1)

- [x] A) (0.6, 0.8)
- [ ] B) (6, 8)

**Respuesta Correcta: A**

¿Cuál es el ángulo entre (1, 0) y (1, 1)?
### Opciones
- [x] A) 45°
- [ ] B) 30°
- [ ] C) 60°
- [ ] D) 90°
### Explicación Pedagógica
**Respuesta Correcta: A**
cosθ = (1·1 + 0·1) / (1 × √2) = 1/√2 → θ = 45°

## Pregunta 7 (Difícil B - Dificultad 5)
**ID:** "CO-MAT-11-GEO-003-v7"
### Enunciado
Un avión en Bogotá viaja con velocidad (200, 100) km/h. Un viento sopla con (-50, 30) km/h. ¿Velocidad resultante?
### Opciones
- [x] A) (150, 130) km/h
- [ ] B) (250, 70) km/h
- [ ] C) (200, 100) km/h
- [ ] D) (150, 70) km/h
### Explicación Pedagógica
**Respuesta Correcta: A**
(200-50, 100+30) = (150, 130)

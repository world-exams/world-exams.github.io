## Pregunta 1 (Original - Dificultad 3)
**ID:** "CO-MAT-11-derivadas-004-v1"
### Enunciado
Para f(x) = x² - 4x + 3, ¿cuál es el valor de x en el vértice?
### Opciones
- [x] A) 2
- [ ] B) 0
- [ ] C) 4
- [ ] D) -2
### Explicación Pedagógica
**Respuesta Correcta: A**
f'(x) = 2x - 4 = 0, x = 2

## Pregunta 4 (Media A - Dificultad 3)
**ID:** "CO-MAT-11-derivadas-004-v4"
### Enunciado
Una empresa manufactura productos con costo C(x) = 100 + 5x. Ingreso R(x) = 20x - 0.5x². ¿En qué cantidad x se maximiza la ganancia?
### Opciones
- [x] A) x = 15
- [ ] B) x = 10
- [ ] C) x = 20
- [ ] D) x = 5
### Explicación Pedagógica
**Respuesta Correcta: A**
Ganancia G(x) = R(x) - C(x) = 20x - 0.5x² - 100 - 5x = -0.5x² + 15x - 100
G'(x) = -x + 15 = 0, x = 15

## Pregunta 7 (Difícil B - Dificultad 5)
**ID:** "CO-MAT-11-derivadas-004-v7"
### Enunciado
Un rectángulo se inscribe en un círculo de radio 5. ¿Cuál es el área máxima?
### Opciones
- [x] A) 50
- [ ] B) 25
- [ ] C) 100
- [ ] D) 75
### Explicación Pedagógica
**Respuesta Correcta: A**
Área = 2xy donde x² + y² = 25. Optimización resulta en A_{max} = 50

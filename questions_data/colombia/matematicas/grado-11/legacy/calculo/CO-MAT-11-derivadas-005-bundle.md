# Pregunta Base: Monotonía de Funciones
## Pregunta 1 (Original - Dificultad 3)
**ID:** "CO-MAT-11-derivadas-005-v1"
### Enunciado
Para f(x) = x², determina dónde es creciente.
### Opciones
- [x] A) Para x > 0
- [ ] B) Para x < 0
- [ ] C) En todos lados
- [ ] D) Nunca
### Explicación Pedagógica
**Respuesta Correcta: A**
f'(x) = 2x > 0 cuando x > 0

## Pregunta 5 (Media B - Dificultad 3)
**ID:** "CO-MAT-11-derivadas-005-v5"
### Enunciado
Para f(x) = x³ - 3x, en qué intervalo es creciente?
### Opciones
- [x] A) (-∞, -1) y (1, ∞)
- [ ] B) (-1, 1)
- [ ] C) En todos lados
- [ ] D) Nunca
### Explicación Pedagógica
**Respuesta Correcta: A**
f'(x) = 3x² - 3 > 0 cuando x² > 1, es decir x < -1 o x > 1

## Pregunta 6 (Difícil A - Dificultad 4)
**ID:** "CO-MAT-11-derivadas-005-v6"
### Enunciado
Para f(x) = x³ - 3x², determina intervalos de crecimiento y decrecimiento.
### Opciones
- [x] A) Crece en (-∞, 0) y (2, ∞); Decrece en (0, 2)
- [ ] B) Crece en (0, 2); Decrece en otros
- [ ] C) Siempre creciente
- [ ] D) Siempre decreciente
### Explicación Pedagógica
**Respuesta Correcta: A**
f'(x) = 3x² - 6x = 3x(x-2)
- f'(x) > 0 cuando x < 0 o x > 2 (crece)
- f'(x) < 0 cuando 0 < x < 2 (decrece)

## Pregunta 7 (Difícil B - Dificultad 5)
**ID:** "CO-MAT-11-derivadas-005-v7"
### Enunciado
Si f'(x) = (x-1)(x+2), determina monotonía completa.
### Opciones
- [x] A) Crece en (-∞, -2) y (1, ∞); Decrece en (-2, 1)
- [ ] B) Crece en (-2, 1)
- [ ] C) Siempre crece
- [ ] D) Nunca crece
### Explicación Pedagógica
**Respuesta Correcta: A**
Analizando signos de f'(x) = (x-1)(x+2):
- Crece cuando (x-1)(x+2) > 0: x < -2 o x > 1
- Decrece cuando (x-1)(x+2) < 0: -2 < x < 1

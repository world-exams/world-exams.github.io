# Pregunta Base: Máximos y Mínimos
## Pregunta 1 (Original - Dificultad 3)
**ID:** "CO-MAT-11-derivadas-006-v1"
### Enunciado
f(x) = -x² + 4x. ¿En qué valor de x alcanza su máximo?
### Opciones
- [x] A) x = 2
- [ ] B) x = 4
- [ ] C) x = 0
- [ ] D) x = -2
### Explicación Pedagógica
**Respuesta Correcta: A**
f'(x) = -2x + 4 = 0 → x = 2

## Pregunta 6 (Difícil A - Dificultad 4)
**ID:** "CO-MAT-11-derivadas-006-v6"
### Enunciado
Costo C(x) = x² - 10x + 100. Mínimo costo en:
### Opciones
- [x] A) x = 5, C = $75
- [ ] B) x = 10, C = $100
- [ ] C) x = 0, C = $100
- [ ] D) x = 5, C = $50
### Explicación Pedagógica
**Respuesta Correcta: A**
C'(x) = 2x - 10 = 0 → x = 5
C(5) = 25 - 50 + 100 = 75

## Pregunta 7 (Difícil B - Dificultad 5)
**ID:** "CO-MAT-11-derivadas-006-v7"
### Enunciado
En Bogotá, ganancia G(x) = -2x² + 80x - 200 (en miles). ¿Cuántas unidades maximizan ganancia?
### Opciones
- [x] A) 20 unidades (ganancia $600.000)
- [ ] B) 40 unidades
- [ ] C) 10 unidades
- [ ] D) 80 unidades
### Explicación Pedagógica
**Respuesta Correcta: A**
G'(x) = -4x + 80 = 0 → x = 20
G(20) = -800 + 1600 - 200 = 600

# Pregunta Base: Definición y Cálculo de Derivadas
## Pregunta 1 (Original - Dificultad 3)
**ID:** "CO-MAT-11-derivadas-002-v1"
### Enunciado
Calcula la derivada de f(x) = 3x² + 2x
### Opciones
- [x] A) 6x + 2
- [ ] B) 3x + 2
- [ ] C) 6x
- [ ] D) 6x + 1
### Explicación Pedagógica
**Respuesta Correcta: A**
f'(x) = d/dx(3x²) + d/dx(2x) = 6x + 2

Derivada de f(x) = x³
### Opciones
- [x] A) 3x²
- [ ] B) x²
- [ ] C) 3x
- [ ] D) 3
### Explicación Pedagógica
**Respuesta Correcta: A**
Regla de potencia: d/dx(xⁿ) = nxⁿ⁻¹
f'(x) = 3x³⁻¹ = 3x²

## Pregunta 7 (Difícil B - Dificultad 5)
**ID:** "CO-MAT-11-derivadas-002-v7"
### Enunciado
Encontrar f'(x) si f(x) = (x³ + 2)²
### Opciones
- [x] A) 6x(x³ + 2)
- [ ] B) 2(x³ + 2)
- [ ] C) 6xµ
- [ ] D) (x³ + 2)
### Explicación Pedagógica
**Respuesta Correcta: A**
Usando la regla de la cadena con u = x³ + 2:
f'(x) = 2(x³ + 2) · 3x² = 6x²(x³ + 2) = 6x(x³ + 2)

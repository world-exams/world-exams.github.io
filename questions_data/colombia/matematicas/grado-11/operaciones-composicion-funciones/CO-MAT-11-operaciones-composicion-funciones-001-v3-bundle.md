## Pregunta 1 (Muy Fácil A - Dificultad 1)

**ID:** `CO-MAT-11-operaciones-composicion-funciones-001-v1`

### Enunciado

Si tenemos dos funciones $f(x)$ y $g(x)$, ¿cómo se define la **Suma** $(f+g)(x)$?

### Opciones

- [x] A) $(f+g)(x) = f(x) + g(x)$
- [ ] B) $(f+g)(x) = f(g(x))$
- [ ] C) $(f+g)(x) = f(x) \cdot g(x)$
- [ ] D) $(f+g)(x) = 0$

### Explicación Pedagógica

Es simplemente sumar las expresiones de ambas funciones para cada valor de $x$.

## Pregunta 2 (Muy Fácil B - Dificultad 1)

**ID:** `CO-MAT-11-operaciones-composicion-funciones-001-v2`

### Enunciado

¿Qué significa la notación $(f \circ g)(x)$?

### Opciones

- [x] A) **Composición** de funciones: $f(g(x))$, es decir, primero aplicas $g$ y al resultado le aplicas $f$
- [ ] B) Multiplicación de funciones: $f(x) \cdot g(x)$
- [ ] C) Coeficiente de funciones
- [ ] D) División de funciones

### Explicación Pedagógica

Se lee "$f$ compuesta con $g$" o "$f$ de $g$ de $x$". Es meter una función dentro de otra.

## Pregunta 3 (Fácil A - Dificultad 2)

**ID:** `CO-MAT-11-operaciones-composicion-funciones-001-v3`

### Enunciado

Si $f(x) = 2x$ y $g(x) = x + 3$, halla $(f \circ g)(x)$.

### Opciones

- [x] A) $f(x+3) = 2(x+3) = 2x + 6$
- [ ] B) $2x + 3$
- [ ] C) $2x^2 + 3$
- [ ] D) $x + 6$

### Explicación Pedagógica

Reemplazamos la $x$ de $f$ por toda la expresión de $g$.

## Pregunta 4 (Fácil B - Dificultad 2)

**ID:** `CO-MAT-11-operaciones-composicion-funciones-001-v4`

### Enunciado

Usando las mismas funciones $f(x) = 2x$ y $g(x) = x + 3$, halla $(g \circ f)(x)$.

### Opciones

- [x] A) $g(2x) = (2x) + 3 = 2x + 3$
- [ ] B) $2x + 6$
- [ ] C) $3x + 2$
- [ ] D) $6x$

### Explicación Pedagógica

Nota que $(f \circ g) \neq (g \circ f)$ en general; la composición no es conmutativa.

## Pregunta 5 (Media A - Dificultad 3)

**ID:** `CO-MAT-11-operaciones-composicion-funciones-001-v5`

### Enunciado

¿Cuál es el dominio de la función producto $(f \cdot g)(x)$?

### Opciones

- [x] A) La **intersección** de los dominios de $f$ y $g$ ($Dom_f \cap Dom_g$)
- [ ] B) La unión de los dominios
- [ ] C) Todos los reales siempre
- [ ] D) Solo los positivos

### Explicación Pedagógica

Para poder multiplicar los resultados, ambas funciones deben "funcionar" (existir) para ese valor de $x$ simultáneamente.

## Pregunta 6 (Media B - Dificultad 3)

**ID:** `CO-MAT-11-operaciones-composicion-funciones-001-v6`

### Enunciado

Si $f(x) = \sqrt{x}$ y $g(x) = x - 4$, ¿cuál es el dominio de $(f \circ g)(x) = \sqrt{x-4}$?

### Opciones

- [x] A) $x \geq 4$ (Intervalo $[4, \infty)$)
- [ ] B) Todos los reales
- [ ] C) $x \geq 0$
- [ ] D) $x \leq 4$

### Explicación Pedagógica

La función compuesta debe ser válida, por tanto $x-4 \geq 0$.

## Pregunta 7 (Difícil A - Dificultad 4)

**ID:** `CO-MAT-11-operaciones-composicion-funciones-001-v7`

### Enunciado

¿Qué es la **Función Inversa** $f^{-1}(x)$?

### Opciones

- [x] A) Una función que "deshace" lo que hace $f$; si $f(a) = b$, entonces $f^{-1}(b) = a$
- [ ] B) $1 / f(x)$
- [ ] C) $-f(x)$
- [ ] D) La derivada de $f$

### Explicación Pedagógica

Gráficamente, es una reflexión respecto a la recta $y = x$. Solo las funciones inyectivas (uno a uno) tienen inversa.

## Pregunta 8 (Difícil B - Dificultad 4)

**ID:** `CO-MAT-11-operaciones-composicion-funciones-001-v8`

### Enunciado

Halla la inversa de $f(x) = 3x - 5$.

### Opciones

- [x] A) $f^{-1}(x) = \frac{x+5}{3}$
- [ ] B) $f^{-1}(x) = 3x + 5$
- [ ] C) $f^{-1}(x) = \frac{x-5}{3}$
- [ ] D) $f^{-1}(x) = \frac{1}{3x-5}$

### Explicación Pedagógica

Pasos: 1) $y = 3x - 5$. 2) Despejar $x$: $y + 5 = 3x \rightarrow x = (y+5)/3$. 3) Cambiar variables: $y = (x+5)/3$.

## Pregunta 9 (Muy Difícil A - Dificultad 5)

**ID:** `CO-MAT-11-operaciones-composicion-funciones-001-v9`

### Enunciado

Si $(f \circ g)(x) = x$ para todo $x$, y $(g \circ f)(x) = x$ para todo $x$, ¿qué podemos concluir sobre $f$ y $g$?

### Opciones

- [x] A) Que son funciones **inversas** entre sí ($g = f^{-1}$)
- [ ] B) Que son iguales ($f = g$)
- [ ] C) Que son constantes
- [ ] D) Que son negativas

### Explicación Pedagógica

Al componerlas se anulan mutuamente y devuelven la variable original (función identidad).

## Pregunta 10 (Muy Difícil B - Dificultad 5)

**ID:** `CO-MAT-11-operaciones-composicion-funciones-001-v10`

### Enunciado

Dada $h(x) = (2x + 1)^3$, ¿cuáles funciones $f$ y $g$ podrían componerla de modo que $h = f \circ g$?

### Opciones

- [x] A) $g(x) = 2x + 1$ (interna) y $f(x) = x^3$ (externa)
- [ ] B) $f(x) = 2x + 1$ y $g(x) = x^3$
- [ ] C) $f(x) = 2x$ y $g(x) = x^3 + 1$
- [ ] D) $f(x) = x + 1$ y $g(x) = (2x)^3$

### Explicación Pedagógica

Primero calculas $2x+1$ (función interna $g$) y luego elevas todo al cubo (función externa $f$).

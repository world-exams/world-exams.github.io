## Pregunta 1 (Muy Fácil A - Dificultad 1)

**ID:** `CO-MAT-7-expresiones-algebraicas-001-v1`

### Enunciado

¿Qué es una **Variable** o incógnita en álgebra?

### Opciones

- [x] A) Una letra (como $x$, $y$, $a$) que representa un número desconocido o que puede cambiar de valor
- [ ] B) Un número que nunca cambia
- [ ] C) El resultado siempre de una suma
- [ ] D) Un signo de puntuación

### Explicación Pedagógica

El álgebra usa letras para generalizar las reglas de la aritmética. $x + 2$ significa "cualquier número más dos".

**ID:** `CO-MAT-7-expresiones-algebraicas-001-v2`

### Enunciado

¿Cómo se escribe en lenguaje algebraico: "El doble de un número"?

### Opciones

- [x] A) $2x$
- [ ] B) $x + 2$
- [ ] C) $x^2$
- [ ] D) $\frac{x}{2}$

Doble significa multiplicar por $2$. Si el número es $x$, su doble es $2x$.

## Pregunta 4 (Fácil B - Dificultad 2)

**ID:** `CO-MAT-7-expresiones-algebraicas-001-v4`

### Enunciado

Si $x = 3$, ¿cuál es el **valor numérico** de la expresión $4x + 1$?

### Opciones

- [x] A) $13$
- [ ] B) $12$
- [ ] C) $7$
- [ ] D) $4$

### Explicación Pedagógica

Reemplazamos la $x$ por $3$: $(4 \times 3) + 1 = 12 + 1 = 13$.

## Pregunta 5 (Media A - Dificultad 3)

**ID:** `CO-MAT-7-expresiones-algebraicas-001-v5`

### Enunciado

¿Qué son **Términos Semejantes**?

### Opciones

- [x] A) Términos que tienen exactamente la misma parte literal (las mismas letras con los mismos exponentes)
- [ ] B) Términos que tienen el mismo coeficiente numérico
- [ ] C) Términos que tienen el mismo signo
- [ ] D) Términos que están uno al lado del otro

### Explicación Pedagógica

Ejemplo: $3x^2$ y $-7x^2$ son semejantes. Solo los términos semejantes se pueden sumar o restar entre sí.

## Pregunta 6 (Media B - Dificultad 3)

**ID:** `CO-MAT-7-expresiones-algebraicas-001-v6`

### Enunciado

Simplifica la expresión sumando términos semejantes: $2a + 3b - a + 2b$.

### Opciones

- [x] A) $a + 5b$
- [ ] B) $3a + 5b$
- [ ] C) $6ab$
- [ ] D) $a + b$

### Explicación Pedagógica

Juntamos las $a$: $(2a - a = a)$. Juntamos las $b$: $(3b + 2b = 5b)$. Resultado: $a + 5b$.

## Pregunta 8 (Difícil B - Dificultad 4)

**ID:** `CO-MAT-7-expresiones-algebraicas-001-v8`

### Enunciado

Calcula el valor numérico de la expresión $2x^2 - 3y$ si $x = -2$ y $y = 4$.

### Opciones

- [x] A) $-4$
- [ ] B) $4$
- [ ] C) $20$
- [ ] D) $-16$

### Explicación Pedagógica

$x^2 = (-2)^2 = 4$. Primer término: $2(4) = 8$. Segundo término: $-3(4) = -12$. Resta: $8 - 12 = -4$.

## Pregunta 9 (Muy Difícil A - Dificultad 5)

**ID:** `CO-MAT-7-expresiones-algebraicas-001-v9`

### Enunciado

Reduce la siguiente expresión: $5(x - 2) - 2(x + 3)$.

### Opciones

- [x] A) $3x - 16$
- [ ] B) $3x + 1$
- [ ] C) $7x - 4$
- [ ] D) $3x - 4$

### Explicación Pedagógica

Distributiva: $5x - 10 - 2x - 6$.
Agrupamos $x$: $5x - 2x = 3x$.
Agrupamos números: $-10 - 6 = -16$.
Resultado: $3x - 16$.

## Pregunta 10 (Muy Difícil B - Dificultad 5)

**ID:** `CO-MAT-7-expresiones-algebraicas-001-v10`

### Enunciado

El perímetro de un rectángulo se expresa como $2 \times (\text{largo} + \text{ancho})$. Si el largo es $2x$ y el ancho es $x + 1$, ¿cuál es la expresión del perímetro?

### Opciones

- [x] A) $6x + 2$
- [ ] B) $3x + 1$
- [ ] C) $4x + 2$
- [ ] D) $6x + 1$

### Explicación Pedagógica

Suma de lados: $2x + (x+1) = 3x + 1$.
Multiplicamos por $2$: $2(3x + 1) = 6x + 2$.

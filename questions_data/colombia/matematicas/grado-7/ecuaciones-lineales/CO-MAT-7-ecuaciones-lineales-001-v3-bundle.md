## Pregunta 1 (Muy Fácil A - Dificultad 1)

**ID:** `CO-MAT-7-ecuaciones-lineales-001-v1`

### Enunciado

¿Qué es una **Ecuación**?

### Opciones

- [x] A) Una igualdad entre dos expresiones algebraicas donde hay al menos un valor desconocido (incógnita) que debemos encontrar
- [ ] B) Una desigualdad que usa el signo mayor que ($>$)
- [ ] C) Una operación sin resultado
- [ ] D) Una lista de números primos

### Explicación Pedagógica

La ecuación es como una balanza equilibrada; lo que hay a un lado del igual ($=$) tiene el mismo valor que lo que hay al otro lado.

**ID:** `CO-MAT-7-ecuaciones-lineales-001-v2`

### Enunciado

¿Cuál es el valor de $x$ en la ecuación: $x + 5 = 12$?

### Opciones

- [x] A) $7$
- [ ] B) $17$
- [ ] C) $5$
- [ ] D) $12$

### Explicación Pedagógica

Buscamos un número que al sumarle $5$ dé $12$. $12 - 5 = 7$.

## Pregunta 3 (Fácil A - Dificultad 2)

**ID:** `CO-MAT-7-ecuaciones-lineales-001-v3`

### Enunciado

Resuelve para $m$: $3m = 15$.

### Opciones

- [x] A) $5$
- [ ] B) $12$
- [ ] C) $45$
- [ ] D) $3$

### Explicación Pedagógica

El $3$ está multiplicando a la $m$, así que para despejar pasamos a dividir: $m = 15 \div 3 = 5$.

## Pregunta 7 (Difícil A - Dificultad 4)

**ID:** `CO-MAT-7-ecuaciones-lineales-001-v7`

### Enunciado

Resuelve: $5x - 2 = 3x + 8$.

### Opciones

- [x] A) $5$
- [ ] B) $3$
- [ ] C) $10$
- [ ] D) $2$

### Explicación Pedagógica

Agrupamos las $x$ a un lado: $5x - 3x = 8 + 2$.
$2x = 10$.
$x = 5$.

## Pregunta 8 (Difícil B - Dificultad 4)

**ID:** `CO-MAT-7-ecuaciones-lineales-001-v8`

### Enunciado

Si el triple de un número disminuido en $4$ es igual a $20$, ¿cuál es el número?

### Opciones

- [x] A) $8$
- [ ] B) $6$
- [ ] C) $24$
- [ ] D) $12$

### Explicación Pedagógica

Ecuación: $3x - 4 = 20$.
$3x = 24$.
$x = 8$.

## Pregunta 9 (Muy Difícil A - Dificultad 5)

**ID:** `CO-MAT-7-ecuaciones-lineales-001-v9`

### Enunciado

Resuelve la ecuación con paréntesis: $2(x + 1) = 3(x - 2)$.

### Opciones

- [x] A) $8$
- [ ] B) $-8$
- [ ] C) $4$
- [ ] D) $3$

### Explicación Pedagógica

Distributiva: $2x + 2 = 3x - 6$.
Reorganizamos: $2 + 6 = 3x - 2x$.
$8 = x$.

## Pregunta 10 (Muy Difícil B - Dificultad 5)

**ID:** `CO-MAT-7-ecuaciones-lineales-001-v10`

### Enunciado

La suma de tres números consecutivos es $33$. ¿Cuáles son los números? (Plantea la ecuación: $x + (x+1) + (x+2) = 33$).

### Opciones

- [x] A) $10, 11, 12$
- [ ] B) $9, 10, 11$
- [ ] C) $11, 12, 13$
- [ ] D) $10, 20, 3$

### Explicación Pedagógica

$3x + 3 = 33$.
$3x = 30$.
$x = 10$.
Los números son $10, 11$ y $12$.

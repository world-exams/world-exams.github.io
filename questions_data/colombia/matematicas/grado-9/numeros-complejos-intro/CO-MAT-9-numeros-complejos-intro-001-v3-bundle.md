## Pregunta 1 (Muy Fácil A - Dificultad 1)

**ID:** `CO-MAT-9-numeros-complejos-intro-001-v1`

### Enunciado

¿Cuál es la unidad imaginaria, representada por la letra $i$?

### Opciones

- [x] A) $i = \sqrt{-1}$
- [ ] B) $i = 1$
- [ ] C) $i = -1$
- [ ] D) $i = 0$

### Explicación Pedagógica

Se inventó para poder sacar raíces cuadradas de números negativos, algo imposible en los Reales.

## Pregunta 2 (Muy Fácil B - Dificultad 1)

**ID:** `CO-MAT-9-numeros-complejos-intro-001-v2`

### Enunciado

¿Cómo se escribe un número complejo en su forma estándar (binómica)?

### Opciones

- [x] A) $a + bi$ (donde $a$ es la parte real y $bi$ es la parte imaginaria)
- [ ] B) $a - b$
- [ ] C) $a \times b$
- [ ] D) $a / b$

### Explicación Pedagógica

Tiene una parte "de verdad" (Real) y una parte "imaginaria". Ej: $3 + 2i$.

## Pregunta 3 (Fácil A - Dificultad 2)

**ID:** `CO-MAT-9-numeros-complejos-intro-001-v3`

### Enunciado

¿Cuánto es $\sqrt{-4}$ en el conjunto de los números complejos?

### Opciones

- [x] A) $2i$
- [ ] B) $-2$
- [ ] C) $2$
- [ ] D) $4i$

### Explicación Pedagógica

$\sqrt{-4} = \sqrt{4 \times -1} = \sqrt{4} \times \sqrt{-1} = 2i$.

## Pregunta 7 (Difícil A - Dificultad 4)

**ID:** `CO-MAT-9-numeros-complejos-intro-001-v7`

### Enunciado

Calcula el valor de $i^2$.

### Opciones

- [x] A) -1
- [ ] B) 1
- [ ] C) $i$
- [ ] D) 0

### Explicación Pedagógica

Si $i = \sqrt{-1}$, entonces $i^2 = (\sqrt{-1})^2 = -1$. ¡Esto es lo mágico de los complejos!

## Pregunta 8 (Difícil B - Dificultad 4)

**ID:** `CO-MAT-9-numeros-complejos-intro-001-v8`

### Enunciado

Multiplica: $(3i) \times (4i)$.

### Opciones

- [x] A) -12
- [ ] B) $12i$
- [ ] C) 12
- [ ] D) $-12i$

### Explicación Pedagógica

$3 \times 4 = 12$. $i \times i = i^2 = -1$.
$12 \times (-1) = -12$.

## Pregunta 9 (Muy Difícil A - Dificultad 5)

**ID:** `CO-MAT-9-numeros-complejos-intro-001-v9`

### Enunciado

Simplifica esta potencia de $i$: $i^{10}$.

### Opciones

- [x] A) -1
- [ ] B) 1
- [ ] C) $i$
- [ ] D) $-i$

### Explicación Pedagógica

Las potencias de $i$ se repiten en ciclo de 4: $i, -1, -i, 1$.
Dividimos exponente entre 4: $10 \div 4$ da residuo 2.
Así que $i^{10} = i^2 = -1$.

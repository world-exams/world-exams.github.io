## Pregunta 1 (Muy Fácil A - Dificultad 1)

**ID:** `CO-MAT-9-ecuacion-cuadratica-formula-general-001-v1`

### Enunciado

¿Cuál es la forma general de una **Ecuación Cuadrática**?

### Opciones

- [x] A) $ax^2 + bx + c = 0$ (donde $a \neq 0$)
- [ ] B) $y = mx + b$
- [ ] C) $ax + b = c$
- [ ] D) $a^2 + b^2 = c^2$

### Explicación Pedagógica

Siempre debe tener un término al cuadrado ($x^2$).

## Pregunta 2 (Muy Fácil B - Dificultad 1)

**ID:** `CO-MAT-9-ecuacion-cuadratica-formula-general-001-v2`

### Enunciado

¿Cuál es la famosa **Fórmula General** (o fórmula del bachiller) para resolver ecuaciones cuadráticas?

### Opciones

- [x] A) $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$
- [ ] B) $x = \frac{-b + \sqrt{a}}{2}$
- [ ] C) $y = mx + b$
- [ ] D) $a^2 + b^2$

### Explicación Pedagógica

¡Apréndetela de memoria! Sirve para resolver CUALQUIER ecuación cuadrática.

## Pregunta 3 (Fácil A - Dificultad 2)

**ID:** `CO-MAT-9-ecuacion-cuadratica-formula-general-001-v3`

### Enunciado

Resuelve la ecuación incompleta: $x^2 - 16 = 0$.

### Opciones

- [x] A) $x = 4$ y $x = -4$
- [ ] B) Solo $x = 4$
- [ ] C) $x = 16$
- [ ] D) $x = 8$

### Explicación Pedagógica

$x^2 = 16 \rightarrow x = \pm \sqrt{16} = \pm 4$.

**ID:** `CO-MAT-9-ecuacion-cuadratica-formula-general-001-v4`

### Enunciado

Resuelve: $x^2 - 5x = 0$ (Factoriza la x).

### Opciones

- [x] A) $x = 0$ y $x = 5$
- [ ] B) Solo $x = 5$
- [ ] C) $x = 0$ y $x = -5$

$x(x - 5) = 0$. Para que el producto sea cero, o la $x=0$ o el paréntesis $(x-5)=0$.

## Pregunta 6 (Media B - Dificultad 3)

**ID:** `CO-MAT-9-ecuacion-cuadratica-formula-general-001-v6`

### Enunciado

Calcula los valores de $x$ para: $x^2 + 2x + 1 = 0$.
(Pista: Es un trinomio cuadrado perfecto).

### Opciones

- [x] A) $x = -1$ (Solución única)
- [ ] B) $x = 1$
- [ ] C) $x = -1$ y $x = 1$
- [ ] D) No tiene solución

### Explicación Pedagógica

$(x + 1)^2 = 0 \rightarrow x + 1 = 0 \rightarrow x = -1$.
El discriminante $b^2 - 4ac = 4 - 4 = 0$.

## Pregunta 8 (Difícil B - Dificultad 4)

**ID:** `CO-MAT-9-ecuacion-cuadratica-formula-general-001-v8`

### Enunciado

¿Cuáles son las soluciones de $x^2 + x - 6 = 0$?
Busca dos números que multiplicados den -6 y sumados 1.

### Opciones

- [x] A) $x = 2$ y $x = -3$
- [ ] B) $x = -2$ y $x = 3$
- [ ] C) $x = 6$ y $x = -1$
- [ ] D) $x = 1$ y $x = -6$

### Explicación Pedagógica

$(x + 3)(x - 2) = 0$.
Si $x+3=0 \rightarrow x=-3$.
Si $x-2=0 \rightarrow x=2$.

## Pregunta 10 (Muy Difícil B - Dificultad 5)

**ID:** `CO-MAT-9-ecuacion-cuadratica-formula-general-001-v10`

### Enunciado

La suma de un número y su cuadrado es 30. ¿Cuál es el número (positivo)?
$x + x^2 = 30$.

### Opciones

- [x] A) 5
- [ ] B) 6
- [ ] C) 4
- [ ] D) 10

### Explicación Pedagógica

$5 + 5^2 = 5 + 25 = 30$. Perfecto.
También funciona -6, pero pedimos el positivo.

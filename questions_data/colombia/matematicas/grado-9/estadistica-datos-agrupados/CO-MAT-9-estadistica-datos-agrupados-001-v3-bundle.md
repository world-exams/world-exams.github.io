## Pregunta 1 (Muy Fácil A - Dificultad 1)

**ID:** `CO-MAT-9-estadistica-datos-agrupados-001-v1`

### Enunciado

¿Cuándo es necesario organizar los datos en **Intervalos de Clase** (Datos Agrupados)?

### Opciones

- [x] A) Cuando la cantidad de datos es muy grande y los valores son muy variados (variable continua), por lo que hacer una tabla simple sería demasiado largo
- [ ] B) Cuando hay pocos datos
- [ ] C) Cuando todos los datos son iguales
- [ ] D) Solo para variables cualitativas

### Explicación Pedagógica

Si tienes las estaturas de 1000 estudiantes, no puedes hacer una fila para cada estatura exacta.

## Pregunta 2 (Muy Fácil B - Dificultad 1)

**ID:** `CO-MAT-9-estadistica-datos-agrupados-001-v2`

### Enunciado

¿Qué es la **Marca de Clase** ($x_i$) de un intervalo?

### Opciones

- [x] A) El punto medio del intervalo (promedio entre el límite inferior y superior), que representa a todos los datos de ese grupo
- [ ] B) El dato más grande
- [ ] C) El dato más pequeño
- [ ] D) La frecuencia

### Explicación Pedagógica

Si el intervalo es $[10, 20)$, la marca de clase es $(10+20)/2 = 15$.

### Enunciado

¿Qué gráfico se utiliza específicamente para representar datos agrupados en intervalos continuos (las barras están pegadas)?

### Opciones

- [x] A) El Histograma
- [ ] B) El Gráfico de Barras (separadas)
- [ ] C) El Gráfico Circular
- [ ] D) El Pictograma

Como los intervalos son continuos (donde termina uno empieza el otro), las barras no tienen espacio entre ellas.

### Enunciado

¿Cómo se calcula el **Rango** de un conjunto de datos?

### Opciones

- [x] A) Dato Mayor menos Dato Menor ($X_{max} - X_{min}$)
- [ ] B) Promedio de datos

## Pregunta 5 (Media A - Dificultad 3)

**ID:** `CO-MAT-9-estadistica-datos-agrupados-001-v5`

### Enunciado

Para calcular la **Media Aritmética** (Promedio) en datos agrupados, ¿qué fórmula usas?

### Opciones

- [x] A) $\bar{x} = \frac{\sum (f_i \cdot x_i)}{N}$ (Sumatoria de frecuencia por marca de clase, dividido entre total de datos)
- [ ] B) $\bar{x} = \frac{\sum x_i}{N}$
- [ ] C) $\bar{x} = \frac{N}{2}$
- [ ] D) Sumar los límites

### Explicación Pedagógica

Multiplicas la frecuencia de cada grupo por su representante (marca de clase) y luego promedias.

## Pregunta 6 (Media B - Dificultad 3)

**ID:** `CO-MAT-9-estadistica-datos-agrupados-001-v6`

### Enunciado

¿Qué es la **Frecuencia Acumulada** ($F_i$)?

### Opciones

- [x] A) La suma de las frecuencias absolutas de todos los intervalos anteriores hasta el actual (cuántos datos hay "hasta aquí")
- [ ] B) La resta de frecuencias
- [ ] C) El porcentaje
- [ ] D) El total de datos siempre

### Explicación Pedagógica

Sirve para encontrar la Mediana y los Cuartiles.

En un Polígono de Frecuencias, ¿qué punto se une con qué punto?

### Opciones

- [ ] B) Se unen los límites inferiores
- [ ] C) Se unen las esquinas de las barras
- [ ] D) Se hace una curva suave

Muestra la tendencia de los datos.

## Pregunta 8 (Difícil B - Dificultad 4)

**ID:** `CO-MAT-9-estadistica-datos-agrupados-001-v8`

### Enunciado

Si en una tabla de frecuencia, la frecuencia relativa acumulada del intervalo 2 es $0.45$ (45%), ¿qué significa?

### Opciones

- [x] A) Que el 45% de los datos son menores o iguales al límite superior del intervalo 2
- [ ] B) Que el 45% de los datos están solo en el intervalo 2
- [ ] C) Que hay 45 datos
- [ ] D) Que el promedio es 45

### Explicación Pedagógica

"Acumulada" significa que incluye todo lo anterior.

## Pregunta 9 (Muy Difícil A - Dificultad 5)

**ID:** `CO-MAT-9-estadistica-datos-agrupados-001-v9`

### Enunciado

¿Cuál es el intervalo donde se encuentra la **Mediana** en una tabla de datos agrupados?

### Opciones

- [x] A) El primer intervalo cuya Frecuencia Acumulada ($F_i$) sea mayor o igual a $N/2$ (la mitad de los datos)
- [ ] B) El intervalo con mayor frecuencia absoluta (Moda)
- [ ] C) El último intervalo
- [ ] D) El primer intervalo

### Explicación Pedagógica

Buscas dónde cae el dato del medio.

## Pregunta 10 (Muy Difícil B - Dificultad 5)

**ID:** `CO-MAT-9-estadistica-datos-agrupados-001-v10`

### Enunciado

¿Qué regla empírica se usa para decidir cuántos intervalos ($k$) debe tener la tabla (Regla de Sturges)?

### Opciones

- [x] A) $k = 1 + 3.322 \log(N)$
- [ ] B) $k = \sqrt{N}$
- [ ] C) $k = N / 2$
- [ ] D) $k = 10$ siempre

### Explicación Pedagógica

Si tienes 100 datos, Sturges sugiere unos 7 u 8 intervalos.

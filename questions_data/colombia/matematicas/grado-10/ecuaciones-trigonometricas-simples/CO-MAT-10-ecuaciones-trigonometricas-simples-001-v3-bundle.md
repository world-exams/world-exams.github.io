## Pregunta 1 (Muy Fácil A - Dificultad 1)

**ID:** `CO-MAT-10-ecuaciones-trigonometricas-simples-001-v1`

### Enunciado

¿Qué es una **Ecuación Trigonométrica**?

### Opciones

- [x] A) Una igualdad que contiene funciones trigonométricas (sen, cos, etc.) y que es verdadera solo para ciertos valores del ángulo incógnita
- [ ] B) Una igualdad que es verdadera para todos los ángulos
- [ ] C) Una fórmula para calcular áreas
- [ ] D) Una línea recta

### Explicación Pedagógica

A diferencia de las identidades (que valen para todo ángulo), las ecuaciones solo tienen un conjunto específico de soluciones.

## Pregunta 2 (Muy Fácil B - Dificultad 1)

**ID:** `CO-MAT-10-ecuaciones-trigonometricas-simples-001-v2`

### Enunciado

¿Cuál es el valor de $x$ en el primer cuadrante si $\operatorname{sen} x = 1$?

### Opciones

- [x] A) $90^{\circ}$ ($\pi/2$ rad)
- [ ] B) $0^{\circ}$
- [ ] C) $180^{\circ}$
- [ ] D) $360^{\circ}$

### Explicación Pedagógica

Es el punto más alto del círculo unitario $(0, 1)$, donde el seno es 1.

## Pregunta 3 (Fácil A - Dificultad 2)

**ID:** `CO-MAT-10-ecuaciones-trigonometricas-simples-001-v3`

### Enunciado

Resuelve la ecuación: $2\cos x - 1 = 0$ para $x$ en el primer cuadrante.

### Opciones

- [x] A) $60^{\circ}$
- [ ] B) $30^{\circ}$
- [ ] C) $45^{\circ}$
- [ ] D) $90^{\circ}$

### Explicación Pedagógica

$2\cos x = 1 \rightarrow \cos x = 1/2$. El coseno vale $1/2$ a los $60^{\circ}$.

## Pregunta 4 (Fácil B - Dificultad 2)

**ID:** `CO-MAT-10-ecuaciones-trigonometricas-simples-001-v4`

### Enunciado

Resuelve: $\tan x = 1$ en el intervalo $[0, 360^{\circ}]$. ¿Cuáles son las soluciones?

### Opciones

- [x] A) $45^{\circ}$ y $225^{\circ}$
- [ ] B) $45^{\circ}$ y $135^{\circ}$
- [ ] C) $90^{\circ}$ y $270^{\circ}$
- [ ] D) Solo $45^{\circ}$

### Explicación Pedagógica

La tangente es positiva en el I y III cuadrantes. $45$ y $180 + 45 = 225$.

## Pregunta 5 (Media A - Dificultad 3)

**ID:** `CO-MAT-10-ecuaciones-trigonometricas-simples-001-v5`

### Enunciado

¿Cuántas soluciones tiene la ecuación $\operatorname{sen} x = 0.5$ en una vuelta completa al círculo ($0 \le x < 360^{\circ}$)?

### Opciones

- [x] A) 2 soluciones (en el I y II cuadrante)
- [ ] B) 1 solución
- [ ] C) 4 soluciones
- [ ] D) Ninguna

### Explicación Pedagógica

El seno es positivo por encima del eje X, lo que toca al círculo en dos puntos.

## Pregunta 6 (Media B - Dificultad 3)

**ID:** `CO-MAT-10-ecuaciones-trigonometricas-simples-001-v6`

### Enunciado

Resuelve: $\operatorname{sen}^2 x = 1$. ¿Cuáles son las soluciones en $[0, 360^{\circ})$?

### Opciones

- [x] A) $90^{\circ}$ y $270^{\circ}$
- [ ] B) $0^{\circ}$ y $180^{\circ}$
- [ ] C) Solo $90^{\circ}$
- [ ] D) $45^{\circ}$ y $135^{\circ}$

### Explicación Pedagógica

$\operatorname{sen} x = \pm 1$. Eso ocurre arriba ($90^{\circ}$) y abajo ($270^{\circ}$).

## Pregunta 7 (Difícil A - Dificultad 4)

**ID:** `CO-MAT-10-ecuaciones-trigonometricas-simples-001-v7`

### Enunciado

¿Cuál es la solución de $\cos x = 2$?

### Opciones

- [x] A) No tiene solución real
- [ ] B) $0$
- [ ] C) $1$
- [ ] D) $360$

### Explicación Pedagógica

Las funciones Seno y Coseno están atrapadas entre $-1$ y $1$. Nunca pueden ser mayores a $1$.

## Pregunta 8 (Difícil B - Dificultad 4)

**ID:** `CO-MAT-10-ecuaciones-trigonometricas-simples-001-v8`

### Enunciado

Resuelve: $2\operatorname{sen} x + \sqrt{3} = 0$. ¿Cuál es una solución posible?

### Opciones

- [x] A) $240^{\circ}$
- [ ] B) $60^{\circ}$
- [ ] C) $120^{\circ}$
- [ ] D) $30^{\circ}$

### Explicación Pedagógica

$\operatorname{sen} x = -\sqrt{3}/2$. Esto ocurre en el III y IV cuadrante. $180 + 60 = 240$.

## Pregunta 9 (Muy Difícil A - Dificultad 5)

**ID:** `CO-MAT-10-ecuaciones-trigonometricas-simples-001-v9`

### Enunciado

Resuelve: $\operatorname{sen} x = \cos x$ para $x$ en el intervalo $[0, 2\pi)$.

### Opciones

- [x] A) $\pi/4$ y $5\pi/4$
- [ ] B) $\pi/2$ y $3\pi/2$
- [ ] C) $\pi/4$ y $3\pi/4$
- [ ] D) $0$ y $\pi$

### Explicación Pedagógica

Equivale a resolver $\tan x = 1$ (dividiendo por $\cos x$).

## Pregunta 10 (Muy Difícil B - Dificultad 5)

**ID:** `CO-MAT-10-ecuaciones-trigonometricas-simples-001-v10`

### Enunciado

¿Cómo expresarías TODAS las soluciones de $\cos x = 1$?

### Opciones

- [x] A) $x = 2k\pi$, donde $k$ es cualquier número entero
- [ ] B) $x = 0$
- [ ] C) $x = \pi + 2k\pi$
- [ ] D) $x = k\pi$

### Explicación Pedagógica

Como la función es periódica, se le debe sumar el número de vueltas completas.

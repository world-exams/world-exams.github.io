## Pregunta 1 (Muy Fácil A - Dificultad 1)

**ID:** `CO-MAT-10-geometria-analitica-distancia-punto-medio-001-v1`

### Enunciado

¿Cuál es la fórmula para hallar la **Distancia** ($d$) entre dos puntos $P_1(x_1, y_1)$ y $P_2(x_2, y_2)$?

### Opciones

- [x] A) $d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}$
- [ ] B) $d = (x_2 + x_1) + (y_2 + y_1)$
- [ ] C) $d = \sqrt{(x_2 + x_1)^2 - (y_2 + y_1)^2}$
- [ ] D) $d = (x_2 - x_1) + (y_2 - y_1)$

### Explicación Pedagógica

Es el Teorema de Pitágoras aplicado a las diferencias de las coordenadas.

## Pregunta 2 (Muy Fácil B - Dificultad 1)

**ID:** `CO-MAT-10-geometria-analitica-distancia-punto-medio-001-v2`

### Enunciado

¿Cuál es la fórmula para hallar las coordenadas del **Punto Medio** ($M$) entre dos puntos?

### Opciones

- [x] A) $M = \left( \frac{x_1 + x_2}{2}, \frac{y_1 + y_2}{2} \right)$
- [ ] B) $M = (x_2 - x_1, y_2 - y_1)$
- [ ] C) $M = (x_1 \cdot x_2, y_1 \cdot y_2)$
- [ ] D) $M = (2x, 2y)$

### Explicación Pedagógica

El punto medio es simplemente el promedio de las coordenadas de los extremos.

## Pregunta 3 (Fácil A - Dificultad 2)

**ID:** `CO-MAT-10-geometria-analitica-distancia-punto-medio-001-v3`

### Enunciado

Calcula el punto medio entre $A(2, 4)$ y $B(6, 8)$.

### Opciones

- [x] A) $(4, 6)$
- [ ] B) $(8, 12)$
- [ ] C) $(2, 2)$
- [ ] D) $(4, 4)$

### Explicación Pedagógica

$x = (2+6)/2 = 4$. $y = (4+8)/2 = 6$. Resultado: $(4, 6)$.

## Pregunta 4 (Fácil B - Dificultad 2)

**ID:** `CO-MAT-10-geometria-analitica-distancia-punto-medio-001-v4`

### Enunciado

¿Cuál es la distancia entre el origen $(0, 0)$ y el punto $(3, 4)$?

### Opciones

- [x] A) $5$ unidades
- [ ] B) $7$ unidades
- [ ] C) $25$ unidades
- [ ] D) $1$ unidad

### Explicación Pedagógica

$\sqrt{3^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = 5$.

## Pregunta 5 (Media A - Dificultad 3)

**ID:** `CO-MAT-10-geometria-analitica-distancia-punto-medio-001-v5`

### Enunciado

Si el punto medio de un segmento es $(3, 5)$ y uno de los extremos es $(1, 2)$, ¿cuál es el otro extremo?

### Opciones

- [x] A) $(5, 8)$
- [ ] B) $(2, 3)$
- [ ] C) $(4, 7)$
- [ ] D) $(6, 10)$

### Explicación Pedagógica

Para que el promedio de $1$ y $x$ sea $3$, $x=5$. Para que el promedio de $2$ y $y$ sea $5$, $y=8$.

## Pregunta 6 (Media B - Dificultad 3)

**ID:** `CO-MAT-10-geometria-analitica-distancia-punto-medio-001-v6`

### Enunciado

Deseas verificar si un triángulo con vértices $A(0,0)$, $B(4,0)$ y $C(2,4)$ es isósceles. Calcula la distancia de los lados $AB$ y $AC$.

### Opciones

- [x] A) $AB = 4$ y $AC = \sqrt{20}$ (No es isósceles respecto a estos dos lados)
- [ ] B) $AB = 4$ y $AC = 4$
- [ ] C) $AB = 2$ y $AC = 2$
- [ ] D) $AB = 0$ y $AC = 0$

### Explicación Pedagógica

$AB = \sqrt{(4-0)^2} = 4$. $AC = \sqrt{(2-0)^2 + (4-0)^2} = \sqrt{4 + 16} = \sqrt{20}$.

## Pregunta 7 (Difícil A - Dificultad 4)

**ID:** `CO-MAT-10-geometria-analitica-distancia-punto-medio-001-v7`

### Enunciado

¿Cómo usarías la fórmula de la distancia para hallar el **Perímetro** de un polígono en el plano cartesiano?

### Opciones

- [x] A) Calculando la distancia entre cada par de vértices consecutivos y sumando todos los resultados
- [ ] B) Multiplicando las coordenadas de los vértices
- [ ] C) Usando solo el punto medio
- [ ] D) Calculando solo la distancia al origen

### Explicación Pedagógica

El perímetro es la suma de las longitudes de los lados, y cada lado es la distancia entre dos puntos.

## Pregunta 8 (Difícil B - Dificultad 4)

**ID:** `CO-MAT-10-geometria-analitica-distancia-punto-medio-001-v8`

### Enunciado

Un círculo tiene centro en $(2, 3)$ y pasa por el punto $(5, 7)$. ¿Cuál es el radio del círculo?

### Opciones

- [x] A) $5$ unidades
- [ ] B) $7$ unidades
- [ ] C) $25$ unidades
- [ ] D) $12$ unidades

### Explicación Pedagógica

El radio es la distancia del centro a cualquier punto del borde: $\sqrt{(5-2)^2 + (7-3)^2} = \sqrt{3^2 + 4^2} = 5$.

## Pregunta 9 (Muy Difícil A - Dificultad 5)

**ID:** `CO-MAT-10-geometria-analitica-distancia-punto-medio-001-v9`

### Enunciado

Si tres puntos $A, B, C$ son colineales, ¿qué relación debe cumplirse entre sus distancias?

### Opciones

- [x] A) La distancia $AB + BC$ debe ser exactamente igual a la distancia $AC$ (siendo $B$ el punto intermedio)
- [ ] B) Forman un triángulo rectángulo
- [ ] C) No tienen relación
- [ ] D) La suma de sus coordenadas es constante

### Explicación Pedagógica

Solo en una línea recta la suma de las partes es igual al todo sin desviaciones.

## Pregunta 10 (Muy Difícil B - Dificultad 5)

**ID:** `CO-MAT-10-geometria-analitica-distancia-punto-medio-001-v10`

### Enunciado

¿Cuál es la distancia entre los puntos $(-1, -2)$ y $(2, 2)$?

### Opciones

- [x] A) $5$ unidades
- [ ] B) $\sqrt{7}$ unidades
- [ ] C) $1$ unidad
- [ ] D) $25$ unidades

### Explicación Pedagógica

$\sqrt{(2 - (-1))^2 + (2 - (-2))^2} = \sqrt{3^2 + 4^2} = 5$.

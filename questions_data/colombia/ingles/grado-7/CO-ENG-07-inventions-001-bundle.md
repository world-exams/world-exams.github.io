## Question 1 (Muy Fácil A - Difficulty 1)

**ID:** `CO-ENG-07-inventions-001-v1`

### Contexto
Identifying essential inventions.

### Enunciado
Which invention allows us to see in the dark without using fire?

### Options
- [x] A) Light bulb
- [ ] B) Television
- [ ] C) Radio
- [ ] D) Bicycle

### Explicación Pedagógica
The **light bulb** (bombilla/foco), invented by Thomas Edison, provides light using electricity.

Difference between Invention and Discovery.
"An **Invention** is a new object, but a **Discovery** is finding something that already ________."

### Enunciado
Choose the word describing existence.

### Options
- [x] A) existed
- [ ] B) created
- [ ] C) bought
- [ ] D) sold

### Explicación Pedagógica

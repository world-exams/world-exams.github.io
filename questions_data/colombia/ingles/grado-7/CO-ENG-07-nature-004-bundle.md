## Question 1 (Muy Fácil A - Difficulty 1)

**ID:** `CO-ENG-07-nature-004-v1`

### Contexto
Vocabulary: The environment.
"To help the Earth, we must **________** our plastic waste."

### Enunciado
Choose the correct verb for dealing with waste.

### Options
- [x] A) recycle
- [ ] B) buy
- [ ] C) eat
- [ ] D) throw

### Explicación Pedagógica
**Recycle** (reciclar) is the correct action to help the environment.

### Enunciado

## Question 10 (Muy Difícil B - Difficulty 5)

**ID:** `CO-ENG-07-nature-004-v10`

### Contexto
Sentence Order.
"use / should / bags / we / reusable"

### Enunciado
Order the words to form a correct recommendation.

### Options
- [x] A) We should use reusable bags.
- [ ] B) Should we use reusable bags.
- [ ] C) Reusable bags we should use.
- [ ] D) We use should reusable bags.

### Explicación Pedagógica
Structure: **Subject (We) + modal (should) + verb (use) + object (reusable bags).**

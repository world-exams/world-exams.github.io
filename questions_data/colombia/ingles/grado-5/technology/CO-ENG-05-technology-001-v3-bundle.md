# Pregunta Base: Parts of a Computer

> **Fuente:** Adapted from standard educational materials for ESL (A1/A2).

---

## Pregunta 1 (Base - Dificultad 1)

**ID:** `CO-ENG-05-technology-001-v1`

### Enunciado
Which device do you use to **click** on icons on the computer screen?

### Opciones
- [x] A) Mouse
- [ ] B) Keyboard
- [ ] C) Printer
- [ ] D) Speaker

### Explicación Pedagógica
El "mouse" (ratón) es el dispositivo diseñado para mover el cursor y hacer clic en elementos de la pantalla. El "keyboard" es para escribir.

## Pregunta 2 (Context Variation - Dificultad 1)

**ID:** `CO-ENG-05-technology-001-v2`

### Enunciado
I use this machine to type my homework and search for information on the internet. What is it?

### Opciones
- [x] A) Computer
- [ ] B) Television
- [ ] C) Radio
- [ ] D) Microwave

### Explicación Pedagógica
La "computer" (computadora) es la herramienta principal para hacer tareas escolares y buscar en internet. Las otras opciones son electrodomésticos para entretenimiento o cocina.

## Pregunta 3 (Understanding - Dificultad 2)

**ID:** `CO-ENG-05-technology-001-v3`

### Enunciado
What is the function of a **screen** (or monitor)?

### Opciones
- [x] A) To show images and text.
- [ ] B) To type letters and numbers.
- [ ] C) To print paper documents.
- [ ] D) To listen to music.

### Explicación Pedagógica
La función de la "screen" (pantalla) es mostrar visualmente la información, como imágenes y texto. Para escribir usamos el teclado, y para escuchar música los parlantes o audífonos.

## Pregunta 4 (Local Context - Dificultad 2)

**ID:** `CO-ENG-05-technology-001-v4`

### Enunciado
In many schools, students use a **tablet** to learn English. What is a tablet?

### Opciones
- [x] A) A small, flat computer with a touchscreen.
- [ ] B) A large machine with a heavy keyboard.
- [ ] C) A type of cell phone without internet.
- [ ] D) A game console for TV only.

### Explicación Pedagógica
Una "tablet" es una computadora portátil pequeña y plana que se maneja tocando la pantalla ("touchscreen"). Es muy común en educación hoy en día.

## Pregunta 5 (Application - Dificultad 3)

**ID:** `CO-ENG-05-technology-001-v5`

### Enunciado
You want to send a written message to your friend using your computer. You should send an:

### Opciones
- [x] A) Email
- [ ] B) E-book
- [ ] C) Error
- [ ] D) Exit

### Explicación Pedagógica
El "Email" (correo electrónico) es el medio estándar para enviar mensajes escritos a través de internet. "E-book" es un libro digital.

## Pregunta 6 (Real World - Dificultad 3)

**ID:** `CO-ENG-05-technology-001-v6`

### Enunciado
If you want to find information about animals online, you use a **search engine** like:

### Opciones
- [x] A) Google
- [ ] B) Word
- [ ] C) Paint
- [ ] D) Excel

### Explicación Pedagógica
Google es un "search engine" (motor de búsqueda) que nos ayuda a encontrar información en la web. Word, Paint y Excel son programas para crear documentos o imágenes, no para buscar en internet.

## Pregunta 7 (Analysis - Dificultad 4)

**ID:** `CO-ENG-05-technology-001-v7`

### Enunciado
Read the sentence:
"Maria needs to **charge** her laptop because the battery is low."

What does she need?

### Opciones
- [x] A) A charger / power cable.
- [ ] B) A new mouse.
- [ ] C) A bigger screen.
- [ ] D) A faster internet.

### Explicación Pedagógica
Si la batería está baja ("battery is low"), se necesita "charge" (cargar) el dispositivo. Para eso, se usa un cargador ("charger").

## Pregunta 8 (Comparison - Dificultad 4)

**ID:** `CO-ENG-05-technology-001-v8`

### Enunciado
Which is safer for your password?

### Opciones
- [x] A) Keeping it secret.
- [ ] B) Telling all your friends.
- [ ] C) Writing it on the classroom board.
- [ ] D) Using "12345" as a password.

### Explicación Pedagógica
La seguridad en internet ("safety") dicta que las contraseñas deben mantenerse en secreto ("secret"). Compartirlas o usar unas muy simples como "12345" es peligroso.

## Pregunta 9 (Synthesis - Dificultad 5)

**ID:** `CO-ENG-05-technology-001-v9`

### Enunciado
You are typing a story, but you made a mistake. Which key do you press to **erase** the letter?

### Opciones
- [x] A) Backspace / Delete
- [ ] B) Enter
- [ ] C) Spacebar
- [ ] D) Shift

### Explicación Pedagógica
La tecla "Backspace" o "Delete" se usa para borrar ("erase") caracteres. "Enter" es para nueva línea, "Spacebar" para espacio y "Shift" para mayúsculas.

## Pregunta 10 (Evaluation - Dificultad 5)

**ID:** `CO-ENG-05-technology-001-v10`

### Enunciado
Your friend says: *"I spend 10 hours every day playing video games."*
Is this a healthy use of technology?

### Opciones
- [x] A) No, it is too much time.
- [ ] B) Yes, it is good for the eyes.
- [ ] C) Yes, technology is always good.
- [ ] D) No, he should play 24 hours.

### Explicación Pedagógica
El uso excesivo de tecnología ("10 hours every day") no es saludable. Puede afectar la vista, el sueño y la actividad física. Es importante tener un equilibrio.

## Question 1 (Standard Application - Difficulty 3)

**ID:** `CO-ENG-11-conditionals-001-PRO-v1`
**Type:** `single`
**ICFES:** Gramatical
**Bloom:** Apply

### Contexto
The weather forecast predicts a severe storm hitting the coast tomorrow morning.

### Enunciado
Select the correct first conditional sentence to express the likely consequence:

### Options
- [x] A) If the storm hits the coast tomorrow, the local authorities will evacuate the residents.
- [ ] B) If the storm will hit the coast tomorrow, the local authorities evacuate the residents.
- [ ] C) If the storm hit the coast tomorrow, the local authorities would evacuate the residents.
- [ ] D) If the storm hits the coast tomorrow, the local authorities would have evacuated the residents.

### Explicación Pedagógica
The first conditional is used for real and highly probable future events. The structure is: `If + Present Simple, Future Simple (will)`. Option A perfectly follows this structure ("hits... will evacuate"). Option B incorrectly puts "will" in the if-clause. Option C is the second conditional (unlikely/hypothetical). Option D incorrectly mixes the first conditional with the third conditional result clause.

### Contexto
John does not have a high enough GPA, so he cannot apply for the prestigious scholarship.

### Enunciado
Which second conditional sentence correctly describes this hypothetical situation?

### Options
- [ ] A) If John has a higher GPA, he could apply for the prestigious scholarship.
- [ ] B) If John had a higher GPA, he can apply for the prestigious scholarship.
- [x] C) If John had a higher GPA, he could apply for the prestigious scholarship.
- [ ] D) If John had had a higher GPA, he could apply for the prestigious scholarship.

### Explicación Pedagógica
The second conditional describes an unreal or hypothetical situation in the present. The structure is: `If + Past Simple, would/could/might + base verb`. Option C uses "had" (Past Simple) and "could apply" correctly. Option A uses Present Simple in the if-clause. Option B mixes past simple with present ability ("can"). Option D uses the Past Perfect (third conditional), which refers to an unreal past, not an unreal present.

**ID:** `CO-ENG-11-conditionals-001-PRO-v5`
**Type:** `single`
**ICFES:** Gramatical + Lexical

### Contexto
He failed to secure the investment last month, so his company is now facing bankruptcy.

### Enunciado
Which mixed conditional sentence accurately links the past event to the present consequence?

### Options

- [ ] B) If he had not failed to secure the investment last month, his company would not have faced bankruptcy now.
- [x] C) If he had secured the investment last month, his company would not be facing bankruptcy now.

Option B uses a third conditional result ("would not have faced"), which refers to the past, contradicting "now". Option D mixes second conditional if-clause with third conditional result.

## Question 6 (Advanced Analysis - Difficulty 4)

**ID:** `CO-ENG-11-conditionals-001-PRO-v6`
**Type:** `single`
**ICFES:** Gramatical
**Bloom:** Analyze

### Contexto
I am giving you this emergency contact number. Use it only in the event that the alarm system fails.

### Enunciado
Choose the sentence that correctly replaces "in the event that" using a conditional conjunction:

### Options
- [ ] A) Unless the alarm system fails, use this emergency contact number.
- [x] B) In case the alarm system fails, use this emergency contact number.
- [ ] C) Provided that the alarm system fails, use this emergency contact number.
- [ ] D) As long as the alarm system fails, use this emergency contact number.

### Explicación Pedagógica
"In case" is used to describe a precaution taken against a possible future problem (e.g., "do this because the alarm might fail"). It is the semantic equivalent of "in the event that". "Unless" means "if not" and would change the meaning to "do not use it if it fails". "Provided that" and "As long as" mean "on the condition that," which sounds like you desire the alarm to fail to use the number, rather than it being a precaution.

## Question 8 (Advanced Analysis - Difficulty 4)

**ID:** `CO-ENG-11-conditionals-001-PRO-v8`
**Type:** `single`
**ICFES:** Gramatical
**Bloom:** Analyze

### Contexto
If I am appointed as the director, I will implement sweeping reforms.

### Enunciado
Identify the correct formal inversion of this conditional sentence:

### Options
- [x] A) Should I be appointed as the director, I will implement sweeping reforms.
- [ ] B) Were I appointed as the director, I will implement sweeping reforms.
- [ ] C) Had I been appointed as the director, I will implement sweeping reforms.
- [ ] D) Do I be appointed as the director, I will implement sweeping reforms.

### Explicación Pedagógica
In formal English, first conditional clauses (If + Present) can be inverted by dropping "If" and starting with "Should". "If I am appointed" becomes "Should I be appointed". Option A correctly applies this rule. Option B ("Were I...") is the inversion for the second conditional, and its result clause would need to be "I would implement". Option C ("Had I been...") is the inversion for the third conditional. Option D is grammatically invalid.

## Question 9 (Advanced Analysis - Difficulty 4)

**ID:** `CO-ENG-11-conditionals-001-PRO-v9`
**Type:** `single`
**ICFES:** Lexical + Comunicativa
**Bloom:** Analyze

### Contexto
"But for the quick intervention of the paramedics, the victim would deeply suffer." (Note: this is slightly unnatural formulation to test grammar correction).

### Enunciado
The phrase "But for" is used in advanced conditionals. What is the most natural correction of the result clause if the intervention happened in the past?

### Options
- [ ] A) the victim would suffer deeply.
- [ ] B) the victim will have deeply suffered.
- [x] C) the victim would have died.
- [ ] D) the victim had been suffering deeply.

### Explicación Pedagógica
"But for + noun phrase" means "If it hadn't been for...". Because the intervention was in the past ("was the intervention of the paramedics"), it sets up an unreal past condition (third conditional). The result clause must use "would have + past participle". Option C provides the correct grammatical structure ("would have died") and fits the extreme context of paramedic intervention. Options A and D use incorrect conditional structures.

Imagine a politician speaking: "If we are elected, taxes will be reduced within our first hundred days in office."

Why did the politician choose the first conditional for this statement rather than the second conditional ("If we were elected, taxes would be reduced...")?

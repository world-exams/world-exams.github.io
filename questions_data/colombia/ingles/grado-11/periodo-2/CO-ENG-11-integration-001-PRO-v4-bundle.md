## Question 1 (Standard Application - Difficulty 3)

**ID:** `CO-ENG-11-integration-001-PRO-v1`

### Contexto
*Integration: Passive Voice & Relative Clauses*
"The novel was written by a mysterious author. The author's true identity has never been revealed."

### Enunciado
Select the most fluid way to combine these sentences using a relative pronoun and passive voice:

### Options
- [ ] A) The novel was written by a mysterious author who true identity has never been revealed.
- [x] B) The novel was written by a mysterious author whose true identity has never been revealed.
- [ ] C) The mysterious author, whose true identity has never been revealed, was written the novel.
- [ ] D) The novel written by a mysterious author, which true identity has never been revealed.

### Explicación Pedagógica
The combination requires possessing the "true identity" (belonging to the author). The possessive relative pronoun is "whose". Option B cleanly structures this: "author whose true identity..." followed by the present perfect passive "has never been revealed." Option A uses "who" incorrectly. Option C breaks the passive voice structure ("was written the novel"). Option D uses "which" (for things) for possession instead of "whose", and is a fragment.

### Contexto
*Integration: Reported Speech & Conditionals*
"If I have time tomorrow, I will help you with the project," Sarah promised John.

### Enunciado
Choose the correct reported speech version of Sarah's conditional promise:

### Options

- [x] C) Sarah promised John that if she had time the next day, she would help him with the project.

### Explicación Pedagógica

Option B incorrectly backshifts the result clause to the third conditional ("would have helped").

- [ ] B) It must have been approved before Friday.
- [x] C) It must be approved before Friday.

Option D incorrectly adds "to" after the modal "must".

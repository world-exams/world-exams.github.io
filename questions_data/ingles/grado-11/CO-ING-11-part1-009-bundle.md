## Pregunta 1 (Original - Dificultad 3)
**ID:** `CO-ING-11-part1-009-v1`

### Enunciado
Which word refers to a machine that can perform tasks automatically?

### Opciones
- [x] A) Robot
- [ ] B) Hammer
- [ ] C) Telescope
- [ ] D) Microscope

### Explicación Pedagógica
A "robot" is defined as a machine capable of carrying out a complex series of actions automatically. Hammers are tools, and telescopes/microscopes are optical instruments.

### Enunciado
What do you use to type on a computer?

### Opciones
- [x] A) Keyboard
- [ ] B) Screen
- [ ] C) Mouse
- [ ] D) Printer

## Pregunta 4 (Media - Dificultad 3)
**ID:** `CO-ING-11-part1-009-v4`

### Enunciado
What is the term for a program designed to damage a computer system?

### Opciones
- [x] A) Malware
- [ ] B) Software
- [ ] C) Hardware
- [ ] D) Firmware

### Explicación Pedagógica
"Malware" (malicious software) is specifically designed to disrupt, damage, or gain unauthorized access to a computer system. Software is a general term, hardware refers to physical parts, and firmware is permanent software.

## Pregunta 5 (Media - Dificultad 3)
**ID:** `CO-ING-11-part1-009-v5`

### Enunciado
In science, what do we call a proposed explanation that can be tested?

### Opciones
- [x] A) Hypothesis
- [ ] B) Conclusion
- [ ] C) Data
- [ ] D) Experiment

### Explicación Pedagógica
A "hypothesis" is a tentative explanation that can be tested through experimentation. A conclusion comes after, data is information collected, and an experiment is the test itself.

## Pregunta 6 (Difícil - Dificultad 5)
**ID:** `CO-ING-11-part1-009-v6`

### Enunciado
Which term describes the process of converting data into a code to prevent unauthorized access?

### Opciones
- [x] A) Encryption
- [ ] B) Deletion
- [ ] C) Compression
- [ ] D) Transmission

### Explicación Pedagógica
"Encryption" is the method of encoding information so that only authorized parties can access it. Compression reduces size, deletion removes data, and transmission sends it.

## Pregunta 7 (Difícil - Dificultad 5)
**ID:** `CO-ING-11-part1-009-v7`

### Enunciado
What is the name of the technology that allows machines to learn from data and improve over time without being explicitly programmed?

### Opciones
- [x] A) Machine Learning
- [ ] B) Virtual Reality
- [ ] C) Cloud Computing
- [ ] D) Blockchain

### Explicación Pedagógica
"Machine Learning" is a subset of AI that focuses on the use of data and algorithms to imitate the way that humans learn, gradually improving its accuracy.

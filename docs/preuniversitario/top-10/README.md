# Top 10 Preuniversitario

Esta carpeta guarda material de trabajo editorial para el modulo preuniversitario.

## Regla
- Solo universidades del Top 10 Colombia
- Solo fuentes oficiales o publicas verificables
- Solo reactivos originales

## Flujo
1. Crear `institution profile`
2. Crear `core blueprint`
3. Crear `packs de 5 .md` por enfoque academico
4. Ensamblar mock institucional

## Backlog operativo
1. Cerrar shortlist reproducible del top 10 con base ICFES Saber Pro.
2. Auditar elegibilidad `exam-first` de cada universidad antes de prometer mock institucional propio.
3. Completar dossier oficial 2026 por universidad elegible.
4. Traducir cada dossier a `core blueprint`.
5. Crear `packs de 5 .md` por enfoque validado.
6. Ensamblar `mock core` compacto y realista.
7. Cuando no exista examen general, mover la universidad a flujo `overlay-first por programa`.

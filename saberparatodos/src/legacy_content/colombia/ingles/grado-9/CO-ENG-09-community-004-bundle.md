## Question 3 (Fácil A - Difficulty 2)

**ID:** `CO-ENG-09-community-004-v3`

### Contexto
Second Conditional (Hypothetical situations).
"If I **________** (be) the mayor, I **________** (improve) the schools."

### Enunciado
Choose the correct pair for the Second Conditional.

### Options
- [x] A) were / would improve
- [ ] B) am / will improve
- [ ] C) be / would improve
- [ ] D) was / improved

### Explicación Pedagógica
Structure: **If + past simple, would + base verb**. 'Were' is used for all persons in formal conditional.

## Question 4 (Fácil B - Difficulty 2)

**ID:** `CO-ENG-09-community-004-v4`

### Contexto
Second Conditional (Results).
"If we **________** (have) more money, we **________** (build) a new park."

### Enunciado
Choose the correct form.

### Options
- [x] A) had / would build
- [ ] B) have / will build
- [ ] C) had / built
- [ ] D) have / would build

### Explicación Pedagógica
If clause: **Past Simple** (had). Result clause: **Would + verb** (would build).

## Question 9 (Muy Difícil A - Difficulty 5)

If more young people got involved in local decisions, our neighborhoods would change positively. However, many teens feel ignored by adults. To fix this, we need platforms where their voices are truly heard."

### Enunciado
Why do many teens not get involved according to the text?

### Options
- [x] A) Because they feel ignored by adults.
- [ ] B) Because they don't care about neighborhoods.
- [ ] C) Because voting is the only option.
- [ ] D) Because they are too busy with homework.

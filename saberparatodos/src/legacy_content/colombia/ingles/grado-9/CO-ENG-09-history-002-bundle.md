Used to (Past states).

### Enunciado
Choose the correct phrase for a past belief that is no longer held.

### Options
- [x] A) used to

## Question 5 (Media A - Difficulty 3)

**ID:** `CO-ENG-09-history-002-v5`

### Contexto
Used to vs. Would.
"We **________** (live) in a small village, and my father **________** (walk) five miles to work every day."

### Enunciado
Choose the most natural combination of phrases for a state and a habit.

### Options
- [x] A) used to live / would walk
- [ ] B) would live / used to walk
- [ ] C) used to live / used to walk
- [ ] D) would live / would walk

### Explicación Pedagógica
"**Used to**" is better for states (living somewhere); "**would**" is great for repeated actions (walking to work).

## Question 7 (Difícil A - Difficulty 4)

**ID:** `CO-ENG-09-history-002-v7`

### Contexto
Identifying incorrect usage.
"Which sentence is **incorrect**?"

### Enunciado
Choose the sentence that uses \'would\' incorrectly for a state.

### Options
- [x] A) My uncle would be very tall when he was young.
- [ ] B) My uncle used to be very tall when he was young.
- [ ] C) My uncle would visit us every summer.
- [ ] D) My uncle used to visit us every summer.

### Explicación Pedagógica
"**Would**" cannot be used to describe past **states** (like being tall); only "used to" works for states.

## Question 1 (Muy Fácil A - Difficulty 1)

### Contexto

### Enunciado
Choose the correct adjective for a low mood.

### Options
- [x] A) depressed
- [ ] B) excited
- [ ] C) hungry
- [ ] D) wealthy

### Explicación Pedagógica
**Depressed** (deprimido) describes a state of severe sadness.

### Options
- [x] A) bored / boring
- [ ] B) boring / bored

### Explicación Pedagógica

## Question 7 (Difícil A - Difficulty 4)

**ID:** `CO-ENG-09-health-002-v7`

### Contexto
Negative Prefixes (Im-/Ir-).
"It is **________**possible to solve every problem alone; seeking help is **________**responsible."

### Enunciado
Choose the correct prefixes for \'possible\' and \'responsible\'.

### Options
- [x] A) im- / ir-
- [ ] B) in- / un-
- [ ] C) un- / im-
- [ ] D) dis- / in-

### Explicación Pedagógica
**Im-** is used before \'p\' (impossible); **Ir-** is used before \'r\' (irresponsible).

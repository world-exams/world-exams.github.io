## Question 1 (Muy Fácil A - Difficulty 1)

**ID:** `CO-ENG-09-tech-002-v1`

### Contexto
Vocabulary.
"A person **________** writes code for software is a programmer."

### Enunciado
Choose the correct relative pronoun for a person.

### Options
- [x] A) who
- [ ] B) which
- [ ] C) where
- [ ] D) when

### Explicación Pedagógica
We use "**who**" as a relative pronoun for people.

Relative Clauses (Non-defining).
"My new laptop, **________** was very expensive, has a great screen."

### Enunciado
Choose the correct pronoun for an extra piece of information (non-defining).

## Question 7 (Difícil A - Difficulty 4)

**ID:** `CO-ENG-09-tech-002-v7`

### Contexto
Omitting the pronoun.
"The app **________** I downloaded really helps me study."

### Enunciado
In which case can you **omit** (delete) the relative pronoun?

### Options
- [x] A) When it is the object of the clause (like 'that' here).
- [ ] B) Always, in every sentence.
- [ ] C) Only if the pronoun is 'who'.
- [ ] D) Never, it is always required.

### Explicación Pedagógica
In defining relative clauses, you can omit the pronoun if it is the **object** of the verb (I downloaded [it]).

### Contexto
Connecting ideas.
"I know a man. His son works for NASA."
"I know a man **________** son works for NASA."

### Enunciado
Choose the correct pronoun to join the sentences.

### Options
- [x] A) whose
- [ ] B) which
- [ ] C) whom
- [ ] D) who

### Explicación Pedagógica
"**Whose**" replaces the possessive adjective (his).

## Question 10 (Muy Difícil B - Difficulty 5)

**ID:** `CO-ENG-09-tech-002-v10`

### Contexto
Grammar check.
"Identify the **correct** sentence:"

### Enunciado

### Options
- [x] A) The person who designed this app is a genius.
- [ ] B) The person which designed this app is a genius.
- [ ] C) The person where designed this app is a genius.
- [ ] D) The person whose designed this app is a genius.

"**Who**" is the only correct pronoun for a person as a subject.
